Platform Specifications

Full technical capabilities, telephony standards, and AI performance metrics.

🧠 Voice Intelligence & LLM

Ultra-Low Latency

End-to-end voice response times under 500ms. Optimized for natural, human-like turn-taking.

Metric: <500ms TTR

Interruptibility (Barge-in)

Users can interrupt the AI at any time. The system stops speaking instantly to listen.

Feature: Full-Duplex

Emotion Detection

Analyzes caller sentiment (Angry, Happy, Neutral) in real-time to adjust tone or escalate.

Output: Sentiment JSON

Multi-Language Support

Native support for 30+ languages with automatic language detection and switching.

Locales: de-DE, en-US, fr-FR...

Context Retention

Remembers details from previous turns and external CRM data injected at runtime.

Memory: Session-Scoped

Answering Machine Detection

Distinguishes between human pickup and voicemail. Leaves voicemails automatically.

Accuracy: >95%

📞 Telephony & SIP

Global Number Provisioning

Instant provisioning of local, mobile, and toll-free numbers in 100+ countries.

SIP Trunking (BYOC)

Bring Your Own Carrier. Connect existing PBX infrastructure via SIP/RTP.

Warm Transfer

Transfer calls to human agents with full context whisper (screen pop) before connecting.

Campaign Dialing

Power dialer for outbound campaigns. Upload CSV lists, set retry logic and time windows.

Call Recording

Stereo recording (dual channel) with automatic transcription and PII redaction options.

DTMF Support

Send and receive keypad tones. Navigate legacy IVR menus automatically.

🛠️ Developer & API

Function Calling

AI triggers client-side functions (e.g., `checkAvailability`) during conversation.

Webhooks

Real-time events for call_start, call_end, transcript_ready, and custom triggers.

WebSocket Audio

Access raw audio streams (Input/Output) via WebSocket for custom processing.

Custom Vocabularies

Boost specific words (brand names, medical terms) for higher recognition accuracy.
