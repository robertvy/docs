## Tests du simulateur web

Le simulateur web offre des tests instantanés de votre agent IA basés sur le navigateur en utilisant la même technologie vocale que les appels de production.

**Fonctionnalités clés** :
- Tester les conversations sans appels téléphoniques
- Surveillance en temps réel de la transcription et des actions
- Pré-remplir les variables de test pour un contexte dynamique
- Journalisation complète dans Conversations

## Démarrage rapide

1. Ouvrez votre agent dans le tableau de bord
2. Le simulateur apparaît sur le côté droit
3. Cliquez sur **Start Test** pour commencer
4. Autorisez l'accès au microphone lorsque demandé
5. Parlez naturellement avec votre agent

## Que tester

- **Accueil** : Vérifiez la prononciation et la clarté
- **Instructions** : Confirmez que l'agent suit vos règles
- **Actions** : Déclenchez des API personnalisées, transferts, réservations
- **Connaissances** : Posez des questions auxquelles votre KB répond
- **Interruptions** : Testez le comportement de barge-in
- **Cas limites** : Silence, parole peu claire, demandes inattendues
