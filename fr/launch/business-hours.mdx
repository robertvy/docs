## Vue d'ensemble

Les horaires d'ouverture contrôlent quand les campagnes sortantes peuvent passer des appels, garantissant que les appels ne se font que pendant les heures appropriées.

Avantages clés :
- **Conformité** : Respectez les exigences légales comme TCPA (8h-21h heure locale aux É.-U.)
- **Expérience client** : Évite d'appeler pendant les heures incommodes
- **Optimisation des coûts** : Concentrez les appels pendant les fenêtres à taux de réponse élevé
- **Gestion des fuseaux horaires** : Respectez l'heure locale dans toutes les régions

## Comment Fonctionnent les Horaires d'Ouverture

Les horaires d'ouverture sont configurés séparément puis assignés aux campagnes. Cela te permet de :
- Réutiliser les mêmes horaires d'ouverture pour plusieurs campagnes
- Gérer les plannings de manière centralisée (ex : "Horaires DACH", "Support 24/7")
- Configurer les fermetures de jours fériés une fois pour toutes les campagnes

    - **Définir par défaut** : Marquez optionnellement comme défaut pour les nouvelles campagnes

  <Step title="Définis le planning hebdomadaire">
    Pour chaque jour de la semaine :
    - Active/Désactive le jour
    - Définis les heures de début et fin (par défaut : 09h00-17h00)
    - Ajoute plusieurs fenêtres horaires par jour si nécessaire

    Configurations courantes :
    - **Lun-Ven 9h-17h** : Horaires de bureau standard
    - **Lun-Ven 9h-18h, Sam 10h-14h** : Disponibilité étendue
    - **Lun-Jeu 8h-20h, Ven 8h-18h** : Horaires commerciaux typiques DACH

    - **Raison** : Description (ex : "Vacances Noël", "Événement Entreprise")
    - **Récurrent** : Répéter annuellement
    - **Fermé** : Pas d'appels à ces dates

### Assigner aux Campagnes

Lors de la création ou modification d'une campagne :
1. Trouve le champ **Horaires d'Ouverture**
2. Sélectionne parmi tes horaires d'ouverture configurés
3. Laissez vide pour fonctionnement 24/7

La campagne ne passera d'appels que pendant les heures configurées dans le fuseau horaire sélectionné.

### Exemples d'Horaires d'Ouverture DACH

<Accordion title="Configurations DACH Courantes">
  **Horaires de Bureau Standard (Allemagne/Autriche/Suisse)**
  - **Lun-Ven 9h00-17h00** : Conservateur, conforme à toutes les réglementations
  - **Lun-Ven 8h00-18h00** : Étendu pour meilleure portée
  - **Lun-Jeu 8h00-20h00, Ven 8h00-18h00** : Fenêtre légale maximale en Allemagne

  **Avec Dérogations de Dates (Jours Fériés Allemands)**
  Fermetures récurrentes courantes :
  - Jour de l'An (1er janv)
  - Vendredi Saint, Lundi de Pâques
  - Fête du Travail (1er mai)
  - Jour de l'Unité Allemande (3 oct)
  - Noël (25-26 déc)

  Configure-les comme dérogations de dates **récurrentes** marquées comme **fermé**.
</Accordion>

### Plannings de Jours Fériés et Spéciaux

Pour les campagnes qui durent plusieurs semaines ou mois :

<Steps>
  <Step title="Ajoute des dérogations de date">
    Configure les fermetures de jours fériés dans tes horaires d'ouverture :
    - Ajoute une dérogation de date pour chaque jour férié
    - Marquez comme **Récurrent** pour répéter annuellement
    - Définis **Fermé** pour bloquer les appels à ces dates
  </Step>

  <Step title="Pause pour événements imprévus">
    Pour les fermetures ad-hoc, naviguez vers campagne active → **Mettre en Pause la Campagne**.
  </Step>

  <Step title="Ajuste les horaires en milieu de campagne">
    Pour modifier les horaires d'appel :
    1. Modifie les horaires d'ouverture assignés (affecte toutes les campagnes qui les utilisent)
    2. Ou Crée de nouveaux horaires d'ouverture et réassignez à la campagne
    3. Les modifications s'appliquent immédiatement aux appels en file d'attente
  </Step>
</Steps>

## Bonnes Pratiques

### Optimisation du Taux de Réponse

Le tableau de bord de campagne fournit une carte thermique hebdomadaire montrant les taux de réponse pour chaque heure, facilitant l'identification des meilleurs moments pour appeler.

<Steps>
  <Step title="Consulte la carte thermique">
    Après tes premiers 50-100 appels, Consulte la carte thermique du taux de réponse du tableau de bord de campagne pour voir quelles heures performent le mieux.
  </Step>

  <Step title="Ajuste les horaires d'ouverture">
    Mets à jour ton configuration d'horaires d'ouverture pour toi concentrer sur les fenêtres horaires performantes montrées dans la carte thermique.
  </Step>

  <Step title="Surveillez les modèles de messagerie vocale">
    Si AMD montre des taux élevés de messagerie vocale (>60%) pendant certaines heures, Évite ces moments—cela signale que les gens sont occupés/indisponibles.
  </Step>
</Steps>

## Conformité & Exigences Légales

<Warning>
**Conformité DACH** : L'Allemagne, l'Autriche et la Suisse ont des réglementations strictes sur les appels sortants. Les appels à froid aux consommateurs sans consentement préalable peuvent entraîner des amendes allant jusqu'à 300 000 € (Allemagne) ou 50 000 € par violation.
</Warning>

### Réglementations Région DACH

<AccordionGroup>
  <Accordion title="Allemagne (UWG & DSGVO)">
    La loi allemande [Gesetz gegen den unlauteren Wettbewerb (UWG)](https://www.gesetze-im-internet.de/englisch_uwg/) régit le télémarketing aux côtés du [RGPD (DSGVO)](https://eur-lex.europa.eu/eli/reg/2016/679/oj/eng).

    **B2C (Consommateurs)** :
    - **Consentement explicite préalable requis** - Les appels à froid aux consommateurs sont interdits sans consentement
    - Aucun consentement présumé n'existe
    - **Horaires d'Ouverture** : 8h à 21h heure locale (jours de semaine recommandés)
    - Fournir un numéro de téléphone dans les annuaires publics ne constitue PAS un consentement

    **B2B (Entreprises)** :
    - Autorisé sous **consentement présumé** si :
      - Le prospect a un intérêt documenté pour ton produit/service
      - L'intérêt doit être concret, pas abstrait
      - Tu peux démontrer un accord présumé d'être appelé
    - **La documentation est critique** - Conserve les registres de l'intérêt démontré

    **Application** :
    - BNetzA (Agence Fédérale des Réseaux) applique les violations
    - Amendes jusqu'à 300 000 € par violation
    - Systèmes de numérotation automatique (robocalls) interdits sans consentement

    **Mise en Œuvre** :
    1. Vérifie que tu as un consentement documenté ou un intérêt commercial légitime
    2. Définis les horaires d'ouverture de 8h à 21h (fuseau horaire Europe/Berlin)
    3. Conserve des registres détaillés de la source et date du consentement
    4. N'appelez jamais les numéros sur les listes d'exclusion
  </Accordion>

  <Accordion title="Autriche (TKG)">
    La loi autrichienne [Telecommunications Act (TKG 2021)](https://ris.bka.gv.at/Dokumente/Erv/ERV_2021_1_190/ERV_2021_1_190.pdf) implémente des règles strictes sur les appels à froid.

    **B2C (Consommateurs)** :
    - **Les appels à froid sont interdits** - Aucun cadre de consentement présumé n'existe
    - Plus strict que l'Allemagne - opt-in explicite requis
    - **Liste d'exclusion ECG** - Registre national ne-pas-appeler

    **B2B (Entreprises)** :
    - Plus souple que B2C mais requiert toujours une relation commerciale justifiable
    - Documentez la connexion commerciale avant d'appeler

    **Horaires d'Ouverture** :
    - Bien que non explicitement mandaté, Suis les standards allemands : 8h - 21h heure locale
    - Évite les weekends pour la prospection à froid

    **Configuration Recommandée** :
    - Utilise le fuseau horaire Europe/Vienna pour les campagnes autrichiennes
    - Configure Lun-Ven 9h - 18h pour conformité conservatrice
    - Vérifie que les contacts ne sont pas sur la liste d'exclusion ECG
  </Accordion>

  <Accordion title="Suisse (UCA)">
    La loi suisse [Unfair Competition Act (UCA)](https://www.fedlex.admin.ch/eli/cc/1988/223_223_223/en) régit le télémarketing.

    **Exigences Clés** :
    - **Exclusion par astérisque** : Toute personne peut enregistrer son numéro avec un astérisque (*) dans l'annuaire téléphonique pour se désinscrire
    - Appeler des numéros exclus est déloyal sauf si **relation commerciale préexistante** existe
    - **Affichage ID appelant requis** : Doit afficher un numéro de téléphone enregistré et autorisé

    **Horaires d'Ouverture** :
    - Pas d'horaires légaux spécifiques, mais bonne pratique : 8h - 20h jours de semaine
    - Respectez le contexte de relation commerciale pour le timing

    **Note RGPD** :
    - La Suisse n'est pas UE, mais la loi suisse fortement influencée par le RGPD
    - Applique des standards similaires de protection des données

    **Configuration Recommandée** :
    - Utilise le fuseau horaire Europe/Zurich
    - Configure Lun-Ven 9h - 18h
    - Assure-toi que ton ID appelant est correctement enregistré
    - Conserve les registres des relations commerciales
  </Accordion>
</AccordionGroup>

### Autres Réglementations Européennes

<AccordionGroup>
  <Accordion title="ePrivacy UE & RGPD">
    Pour les campagnes ciblant d'autres pays de l'UE, conformez-vous à la [Directive ePrivacy (2002/58/EC)](https://eur-lex.europa.eu/eli/dir/2002/58/oj/eng) et au [RGPD (2016/679)](https://eur-lex.europa.eu/eli/reg/2016/679/oj/eng) :

    - **Opt-in préalable requis** pour les appels marketing dans la plupart des pays de l'UE
    - **Horaires d'Ouverture** : Pas de standard UE, mais respectez les normes culturelles
    - **Recommandé** : Lun-Ven 9h - 18h heure locale
    - Évite les heures de déjeuner (12h-14h dans de nombreux pays, surtout France, Espagne, Italie)
  </Accordion>
</AccordionGroup>

### Réglementations Internationales

<AccordionGroup>
  <Accordion title="TCPA É.-U.">
    Pour les campagnes ciblant les consommateurs américains, conformez-vous au [Telephone Consumer Protection Act (47 USC § 227)](https://www.law.cornell.edu/uscode/text/47/227) :

    - **Restrictions Horaires** : 8h - 21h heure locale du destinataire
    - S'applique aux appels marketing/ventes B2C
    - Exemptions : Appels transactionnels, B2B, consentement préalable
    - **Amendes** : Jusqu'à 43 792 $ par violation
    - [Règles TCPA FCC](https://www.fcc.gov/sites/default/files/tcpa-rules.pdf)

    **Configuration** : Utilise 8h - 21h avec fuseau horaire du contact (America/New_York, America/Chicago, etc.)
  </Accordion>

  <Accordion title="Canada (CASL)">
    La législation canadienne [Anti-Spam Legislation (CASL)](https://fightspam.gc.ca/) régit les messages électroniques commerciaux :

    - Consentement préalable requis pour les appels commerciaux
    - Les réglementations provinciales varient
    - Bonne pratique : 8h - 21h heure locale
    - Le Québec a des règles supplémentaires de protection des consommateurs
    - Appliqué par le [CRTC](https://crtc.gc.ca/eng/internet/anti.htm)
  </Accordion>

  <Accordion title="Australie (Do Not Call Register Act)">
    La loi australienne [Do Not Call Register Act 2006](https://www.legislation.gov.au/C2006A00088/latest/text) établit des règles strictes de télémarketing :

    - **Restrictions Horaires** : Lun-Ven 9h - 20h, Sam 9h - 17h (pas d'appels le dimanche)
    - Basé sur l'heure locale du destinataire
    - Amendes lourdes pour violations
    - Doit vérifier contre le [Do Not Call Register](https://www.donotcall.gov.au/)
    - Géré par ACMA (Australian Media and Communications Authority)
  </Accordion>
</AccordionGroup>
