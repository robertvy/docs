Une fois votre agent en direct, quelques habitudes simples le maintiendront en bon fonctionnement et fourniront de bons résultats.

## Surveiller les Performances

Vérifiez régulièrement pour détecter les problèmes rapidement :

**Examinez les Conversations** : Écoutez quelques appels chaque semaine dans **Conversations**. Vérifiez que votre agent gère les appels comme prévu et cherchez des modèles dans les échecs ou transferts.

**Consultez le Tableau de Bord** : Examinez le volume d'appels et les taux de succès dans le **Tableau de Bord** de campagne. Surveillez les baisses soudaines de taux de réponse ou de réussite des objectifs. Utilisez la carte thermique du taux de réponse pour optimiser les heures d'appel.

## Maintenez à Jour

Votre agent performe mieux avec des informations à jour :

<Steps>
  <Step title="Mettez à jour régulièrement les connaissances">
    Quand votre produit, tarifs ou politiques changent, mettez immédiatement à jour la base de connaissances de votre agent pour qu'il donne des informations précises.
  </Step>

  <Step title="Affinez les instructions au besoin">
    Si vous remarquez que votre agent a des difficultés avec certains scénarios, mettez à jour les instructions pour les gérer mieux. Testez les modifications avant de déployer.
  </Step>

  <Step title="Examinez et ajustez les horaires d'ouverture">
    Utilisez l'analytique de campagne pour trouver les meilleurs moments d'appel, puis mettez à jour votre configuration d'horaires d'ouverture pour vous concentrer sur les fenêtres performantes.
  </Step>
</Steps>

## Gérez les Problèmes Rapidement

Quand quelque chose ne va pas :

1. **Vérifiez les modifications récentes** : Avez-vous récemment mis à jour les instructions, connaissances ou intégrations ?
2. **Examinez les journaux d'erreur** : Consultez **Conversations** pour les appels échoués et messages d'erreur
3. **Testez le flux** : Utilisez le test web pour reproduire le problème
4. **Corrigez et vérifiez** : Effectuez la correction, puis testez à nouveau avant de le considérer résolu

## Habitudes de Succès Simples

<AccordionGroup>
  <Accordion title="Écoutez les appels hebdomadairement">
    Choisissez 5-10 conversations aléatoires chaque semaine et écoutez-les. Cela vous garde connecté à la façon dont les clients vivent votre agent et vous aide à repérer les opportunités d'amélioration.
  </Accordion>

  <Accordion title="Suivez ce qui compte">
    Concentrez-vous sur les métriques liées aux résultats commerciaux :
    - Taux de réponse (les gens décrochent-ils ?)
    - Réussite des objectifs (votre agent réussit-il ?)
    - Tendances du volume d'appels (l'utilisation augmente ou diminue ?)
  </Accordion>

  <Accordion title="Testez avant de déployer les modifications">
    Testez toujours les modifications d'instructions ou de connaissances dans le simulateur web avant de mettre à jour votre agent de production. De petites modifications peuvent avoir des effets inattendus.
  </Accordion>

  <Accordion title="Maintenez les listes de contacts propres">
    Supprimez les numéros qui échouent systématiquement ou sont sur les listes ne-pas-appeler. Cela améliore les performances de campagne et vous maintient conforme.
  </Accordion>
</AccordionGroup>

## Quand Demander de l'Aide

Contactez le support si vous rencontrez :
- Échecs d'appels constants ou problèmes de qualité audio
- Baisses de performance soudaines que vous ne pouvez expliquer
- Questions sur la mise à l'échelle ou fonctionnalités avancées
- Problèmes d'intégration avec systèmes externes
