## Authentification

Tous les endpoints API nécessitent une authentification à l'aide d'une clé API transmise dans l'en-tête `X-API-Key` :

```bash
curl -H "X-API-Key: your-api-key" https://api.itellico.ai/v1/accounts/current
```

<Info>
Découvrez comment créer et gérer les clés API dans la [documentation sur les clés API](/accounts/api-keys).
</Info>

## Ressources Principales

L'API est organisée autour de ces ressources principales :

- **Comptes** - Gérez votre compte et vos sous-comptes
- **Agents** - Créez et configurez des agents vocaux IA
- **Fournisseurs** - Accédez aux modèles, transcripteurs et voix disponibles
- **Numéros de Téléphone** - Gérez les numéros de téléphone pour les appels entrants/sortants
- **Trunks SIP** - Configurez les trunks BYOC (Bring Your Own Carrier)
- **Conversations** - Accédez à l'historique et aux détails des conversations
- **Analyses** - Suivez les métriques d'utilisation et les données de performance
