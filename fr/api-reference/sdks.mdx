## Obtenir Votre Clé API

Avant d'utiliser les SDKs, vous aurez besoin d'une clé API depuis votre tableau de bord itellicoAI.

<Steps>
  <Step title="Accédez à Développeurs">
    Rendez-vous sur votre [tableau de bord itellicoAI](https://app.itellico.ai) et accédez à **Développeurs** depuis la barre latérale.
  </Step>

  <Step title="Allez dans Clés API">
    Cliquez sur **Clés API** dans la section Développeurs.
  </Step>

  <Step title="Créez votre clé">
    Cliquez sur le bouton **Créer une Clé API** dans le coin supérieur droit.

    <Frame>
      <div className="block dark:hidden">
        <img src="/images/light.png" alt="Tableau de bord des clés API montrant comment créer une nouvelle clé API" />
      </div>
      <div className="hidden dark:block">
        <img src="/images/dark.png" alt="Tableau de bord des clés API montrant comment créer une nouvelle clé API" />
      </div>
    </Frame>
  </Step>

  <Step title="Copiez et utilisez avec votre SDK">
    Copiez la clé API générée et stockez-la en toute sécurité. Utilisez cette clé lors de l'initialisation du SDK.
  </Step>
</Steps>

<Warning>
Ne partagez jamais votre clé API et ne la commitez pas dans le contrôle de version. Utilisez des variables d'environnement pour stocker votre clé en toute sécurité.
</Warning>

## Comparaison Rapide

### Créer un Agent

<CodeGroup>
```python Python SDK
from itellicoai import Itellicoai

# Initialize client
client = Itellicoai(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    account_id="your-account-id",
    name="Customer Support Agent",
    model={
        "provider": "openai",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "EXAVITQu4vr4xnSDxMaL"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2:general",
        "language": "en-US"
    },
    initial_message={
        "mode": "fixed_message",
        "message": "Hello! How can I assist you today?",
        "delay_ms": 1000
    },
    max_duration_seconds=1800,
    tags=["support", "customer-service"]
)

print(f"Created agent: {agent.id}")
```

```typescript TypeScript SDK
import { Itellicoai } from 'itellicoai';

// Initialize client
const client = new Itellicoai({
  apiKey: 'your-api-key',
});

// Create an agent
const agent = await client.agents.create({
  accountId: 'your-account-id',
  name: 'Customer Support Agent',
  model: {
    provider: 'openai',
    model: 'gpt-4o-mini',
  },
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
  },
  transcriber: {
    provider: 'deepgram',
    model: 'nova-2:general',
    language: 'en-US',
  },
  initialMessage: {
    mode: 'fixed_message',
    message: 'Hello! How can I assist you today?',
    delayMs: 1000,
  },
  maxDurationSeconds: 1800,
  tags: ['support', 'customer-service'],
});

console.log(`Created agent: ${agent.id}`);
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {"provider": "azure", "model": "gpt-4o-mini"}
  }'
```
</CodeGroup>
