## Comment fonctionnent les agents

Un agent itellicoAI est une configuration IA unique qui peut être déployée sur plusieurs canaux simultanément. Le même agent peut :
- Gérer les appels téléphoniques entrants
- Passer des appels sortants dans des campagnes
- Être intégré sur votre site web en tant que widget
- Utiliser différents messages d'accueil pour les scénarios entrants et sortants

Vous n'avez pas nécessairement besoin de créer des agents "entrants" ou "sortants" séparés - le même agent peut être déployé sur plusieurs canaux.

## Déploiement d'appels téléphoniques

Déployez votre agent pour gérer les conversations téléphoniques.

### Appels entrants

<Steps>
  <Step title="Connectez un numéro de téléphone">
    Connectez un numéro de téléphone à votre agent via un tronc SIP
  </Step>
  <Step title="Configurez le message d'accueil">
    Configurez votre message d'accueil entrant pour les appels entrants
  </Step>
  <Step title="Passez en direct">
    Votre agent est prêt à répondre lorsque les clients appellent
  </Step>
</Steps>

**Meilleur pour :**
- Lignes de support client
- Demandes de statut de commande
- Lignes de support technique
- Lignes d'information générale
- Planification de rendez-vous

**Exemple de message d'accueil :**
```text wrap
Merci d'avoir appelé le support Acme. Comment puis-je vous aider aujourd'hui ?
```

### Appels sortants (Campagnes)

<Steps>
  <Step title="Créez une campagne">
    Créez une nouvelle campagne et assignez votre agent
  </Step>
  <Step title="Ajoutez des contacts">
    Téléchargez votre liste de contacts et ajoutez des contacts à la campagne
  </Step>
  <Step title="Configurez le message d'accueil sortant">
    Configurez la substitution du message d'accueil sortant
  </Step>
  <Step title="Lancez la campagne">
    Démarrez votre campagne et surveillez les résultats
  </Step>
</Steps>

**Meilleur pour :**
- Qualification de prospects
- Rappels de rendez-vous
- Enquêtes de feedback client
- Rappels de paiement
- Appels de suivi

**Exemple de message d'accueil :**
```text wrap
Bonjour, c'est Alex d'Acme Software. Vous avez récemment demandé des informations sur notre plan Entreprise. Avez-vous un moment pour discuter ?
```

<Note>
Le même agent peut gérer les appels entrants et sortants. Utilisez la fonctionnalité **substitution de message d'accueil sortant** pour personnaliser la façon dont l'agent se présente lors des appels sortants.
</Note>

## Déploiement de widget Web

Intégrez votre agent sur votre site web pour des interactions basées sur le navigateur.

<Steps>
  <Step title="Créez un widget web">
    Créez un nouveau widget web dans le tableau de bord
  </Step>
  <Step title="Assignez votre agent">
    Sélectionnez quel agent utiliser pour ce widget
  </Step>
  <Step title="Personnalisez l'apparence">
    Ajustez les couleurs, la position et le branding
  </Step>
  <Step title="Intégrez sur votre site web">
    Copiez le code d'intégration et ajoutez-le à votre site
  </Step>
  <Step title="Testez et passez en direct">
    Les visiteurs peuvent maintenant utiliser la voix ou le chat textuel
  </Step>
</Steps>

**Meilleur pour :**
- Support en direct sur le site web
- Questions pré-vente
- Capture de prospects
- Démonstrations de produits interactives
- Assistance web 24h/24 et 7j/7

**Fonctionnalités uniques :**
- Voix ET chat textuel dans une seule interface
- Aucun numéro de téléphone requis
- Persiste lors de la navigation entre les pages
- Peut basculer entre voix et texte en cours de conversation

## Configuration du message d'accueil

Comme les agents peuvent être utilisés pour les appels entrants et sortants, vous pouvez configurer les messages d'accueil séparément :

### Message d'accueil entrant
Utilisé lorsque les clients appellent votre numéro de téléphone ou démarrent un chat web.

```text wrap
Bonjour ! Merci d'avoir appelé le support Acme. Comment puis-je vous aider aujourd'hui ?
```

### Message d'accueil sortant (Substitution)
Utilisé lorsque l'agent passe des appels sortants dans des campagnes.

```text wrap
Bonjour, c'est Alex d'Acme. Vous avez récemment téléchargé notre guide. Avez-vous un moment pour discuter de la façon dont nous pouvons vous aider ?
```

<Tip>
Configurez la substitution du message d'accueil sortant dans **Capacités → Message d'accueil → Substitution sortante**. Si elle n'est pas configurée, l'agent utilise le message d'accueil entrant pour tous les appels.
</Tip>
