## Serveurs MCP (Bêta)

Les serveurs Model Context Protocol (MCP) vous permettent d'étendre vos agents IA avec des outils et sources de données externes.

<Note>
**Cette fonctionnalité est actuellement en bêta avec un accès limité.**

Documentation complète à venir prochainement.
</Note>

---

## Disponibilité

L'intégration des serveurs MCP est actuellement disponible pour certains comptes sélectionnés pendant notre programme bêta.

Si vous êtes intéressé par un accès anticipé à cette fonctionnalité, veuillez contacter notre équipe de support.
