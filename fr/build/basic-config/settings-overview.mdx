## Onglet Modèles

Sélectionnez et configurez les technologies IA qui alimentent votre agent.

### Modèle (LLM)
Choisissez le modèle de langage pour le raisonnement.

**Fournisseurs :**
- Azure OpenAI
- OpenAI
- Anthropic
- Groq

**Fonctionnalités :**
- Sélection de modèle avec badges d'hébergement UE
- Contrôle de température

[En savoir plus sur les modèles IA →](/fr/build/voice-speech/choose-ai-model)

### Voix
Sélectionnez comment votre agent sonne.

**Fournisseurs :**
- ElevenLabs
- Azure
- Cartesia

**Fonctionnalités :**
- Bibliothèque de voix avec aperçus
- Configuration vocale (varie selon le fournisseur)

[En savoir plus sur la sélection de voix →](/fr/build/voice-speech/select-voice)

### Transcripteur
Configurez le transcripteur.

**Fournisseurs :**
- Deepgram
- Azure Speech

**Fonctionnalités :**
- Sélection de langue et support multilingue
- Amplification de mots-clés

[En savoir plus sur la transcription →](/fr/build/voice-speech/transcriber)


### Effets sonores
Ajoutez de l'audio ambiant aux conversations.
- Ambiance de fond (bureau, café, rue)

[En savoir plus sur le son ambiant →](/fr/build/voice-speech/ambient-sound)

## Onglet Capacités

Configurez ce que votre agent peut faire et comment il se comporte dans les conversations.

### Instructions (prompt système)
Définissez la personnalité, les connaissances et le flux conversationnel de votre agent.
- Éditeur compatible Jinja avec coloration syntaxique
- Insertion de variables
- Modèles de prompts et conseils d'écriture

## Conseils de navigation

### Utilisation du panneau de configuration

<Steps>
  <Step title="Configurer les modèles">
    Sélectionnez le modèle IA, la voix et le transcripteur
  </Step>
  <Step title="Configurer les capacités">
    Configurez les instructions, salutations et connaissances
  </Step>
  <Step title="Affiner les opérations">
    Ajustez les paramètres VAD et timeout après test
  </Step>
  <Step title="Configurer les analytics">
    Définissez des objectifs pour la mesure
  </Step>
</Steps>

### Comportement de sauvegarde

Certaines sections (comme la sélection de modèle) sauvegardent automatiquement. D'autres nécessitent de cliquer sur le bouton Enregistrer. Les modifications s'appliquent uniquement aux appels futurs, pas aux appels en cours.
