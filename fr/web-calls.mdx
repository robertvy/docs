## Vue d'ensemble

Ajoutez des conversations vocales et chat alimentées par l'IA à votre site web avec le widget web itellicoAI. Les visiteurs peuvent parler à votre agent IA en utilisant leur voix ou taper des messages dans une interface de chat - le tout dans un seul widget personnalisable qui s'intègre sur n'importe quel site.

- **Apparence** – Choisissez les couleurs, la mise en page, l'avatar et les animations
- **Contenu** – Personnalisez tous les messages et étiquettes dans plusieurs langues
- **Fonctionnalités** – Basculez les modes voix/chat, transcriptions et collecte de feedback
- **Actions** – Ajoutez des boutons cliquables pour les liens ou numéros de téléphone
- **Confidentialité** – Configurez les bannières de consentement et texte légal
- **Domaines** – Restreignez où le widget peut être intégré

## Installation

Ajouter le widget à votre site est simple :

1. Créez et configurez votre widget dans le tableau de bord itellicoAI
2. Copiez le script d'intégration depuis l'onglet Exporter
3. Collez-le avant la balise de fermeture `</body>` sur votre site web
4. Le widget apparaît automatiquement sur votre site

```html
<!-- Collez ceci avant </body> -->
<script src="https://widget.itellico.ai/embed/YOUR-WIDGET-ID.js"></script>
```

<Note>
Fonctionne avec n'importe quelle plateforme de site web : WordPress, Shopify, Webflow, Wix, Squarespace, HTML personnalisé, et plus encore.
</Note>
