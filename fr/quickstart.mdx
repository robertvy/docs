## Étape 2 : Configurez votre agent

### Définir les instructions

Définissez comment votre agent doit se comporter :

1. Développez **Capacités** et cliquez sur **Instructions**
2. Utilisez ce modèle comme point de départ :

```
Vous êtes la voix professionnelle de [Nom de votre entreprise].

Vos objectifs :
- Fournir des informations utiles et précises
- Résoudre les demandes simples de manière autonome
- Escalader les problèmes complexes vers des agents humains

Style de communication :
- Professionnel mais convivial
- Gardez les réponses sous 30 secondes
- Fournissez des conseils clairs et actionnables
```

### Configurer le message d'accueil

Configurez la première impression :

1. Sous **Capacités**, sélectionnez **Message d'accueil**
2. Choisissez **L'agent IA initie avec un message fixe**
3. Saisissez votre message d'accueil :

```
Bonjour ! Merci de contacter [Nom de l'entreprise]. Comment puis-je vous aider aujourd'hui ?
```

4. Définissez le délai à **0,5 secondes** pour un rythme naturel

### Sélectionner le modèle IA et la voix

Choisissez la technologie qui alimente votre agent :

1. Développez **Modèles** et cliquez sur **Modèle**
2. Sélectionnez **Azure OpenAI** avec **GPT-4.1 Mini** (recommandé)
3. Cliquez sur **Voix** et choisissez **ElevenLabs**
4. Prévisualisez les voix et sélectionnez-en une correspondant à votre marque
5. Dans **Transcripteur**, sélectionnez **Deepgram Nova-3 General** avec votre langue préférée ou multilingue

## Étape 3 : Testez votre agent

Avant de passer en production, testez votre agent :

1. Cliquez sur **Appel test** (coin supérieur droit)
2. Cliquez sur **Démarrer un appel web**

  <div className="block dark:hidden">
    <img src="/images/web-test/web-test-light.png" alt="Dialogue Tester l'agent montrant Type de test avec onglets Appel Web et Appel téléphonique, section Paramètres d'appel web avec description des autorisations micro et audio, et bouton Démarrer un appel web" />
  </div>
  <div className="hidden dark:block">

3. Accordez les autorisations du microphone lorsque demandé
4. Essayez différents scénarios :
   - "Bonjour, j'ai besoin d'aide avec mon compte"
   - "Pouvez-vous me parler de vos services ?"
   - "J'aimerais parler à quelqu'un"
5. Évaluez les réponses pour la clarté et le ton

<Note>
Si les réponses semblent incorrectes, ajustez vos instructions et testez à nouveau. L'itération est la clé d'un excellent agent.
</Note>

## Étape 4 : Déployez votre agent

Choisissez comment vous souhaitez déployer votre agent :

<Tabs>
  <Tab title="Appels téléphoniques">
    **Rendez votre agent disponible par téléphone**

    1. Accédez à **Téléphonie → Troncs SIP** dans le tableau de bord
    2. Cliquez sur **Créer un tronc SIP** et configurez les paramètres de votre fournisseur de téléphonie
    3. Allez dans **Téléphonie → Numéros de téléphone**
    4. Cliquez sur **Créer un numéro de téléphone** et liez-le à votre tronc SIP
    5. Assignez votre agent au numéro de téléphone

    Votre agent est maintenant en ligne et peut gérer les appels entrants !

    <Note>
    Vous n'avez pas de tronc SIP ? Envoyez un e-mail à [support@itellico.ai](mailto:support@itellico.ai) pour demander la fourniture d'un numéro de téléphone.
    </Note>

    <Card title="Guide d'intégration téléphonique" icon="phone" href="/fr/phone-calls">
      En savoir plus sur la configuration du tronc SIP et le déploiement téléphonique
    </Card>
  </Tab>

  <Tab title="Widget Web">
    **Intégrez votre agent sur votre site web**

    1. Accédez à **Web → Widgets**
    2. Cliquez sur **Créer un widget** et sélectionnez votre agent
    3. Personnalisez l'apparence et le comportement
    4. Copiez le script d'intégration depuis l'onglet **Exporter**
    5. Collez-le avant `</body>` sur votre site web

    ```html
    <script src="https://widget.itellico.ai/embed/YOUR-WIDGET-ID.js"></script>
    ```

    Votre agent est maintenant en ligne sur votre site web avec voix et chat !

    <Card title="Guide de configuration du widget" icon="globe" href="/fr/web-calls">
      En savoir plus sur la personnalisation du widget
    </Card>
  </Tab>
</Tabs>
