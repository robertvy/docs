## Informations du compte

### Nom du compte

Modifiez le nom d'affichage de votre compte :

1. Allez dans **Compte → Paramètres**
2. Mettez à jour le champ **Nom d'affichage**
3. Cliquez sur **Enregistrer**

<Tip>
Utilisez des noms descriptifs comme "Acme Corp - Équipe ventes" au lieu de noms génériques comme "Compte 1".
</Tip>

### Logo du compte

Téléchargez un logo pour personnaliser votre compte :

1. Allez dans **Compte → Paramètres**
2. Cliquez sur **Télécharger un logo**
3. Sélectionnez un fichier image
4. Cliquez sur **Enregistrer**

**Exigences :**
- Format : PNG ou JPEG
- Taille recommandée : 400×400px
- Taille de fichier maximale : 2MB

### ID du compte

L'identifiant unique (UUID) de votre compte est affiché sur la page des paramètres. Utilisez-le pour les requêtes API et les tickets de support.

**Pour copier :**
1. Allez dans **Compte → Paramètres**
2. Trouvez le champ **ID du compte**
3. Cliquez sur le bouton de copie

<Note>
L'ID du compte ne peut pas être modifié. Il est automatiquement attribué lors de la création du compte.
</Note>
