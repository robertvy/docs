## Outils Disponibles

### SearchItellicoAi

Recherchez dans la base de connaissances itellicoAI pour trouver des informations pertinentes, des exemples de code, des références API et des guides.

**Utilisez cet outil lorsque vous avez besoin de :**
- Répondre à des questions sur itellicoAI
- Trouver une documentation spécifique
- Comprendre le fonctionnement des fonctionnalités
- Localiser les détails d'implémentation

**Retourne :**
- Contenu contextuel avec titres
- Liens directs vers les pages de documentation

---

## Connexion de Votre Application IA

<Steps>
  <Step title="Obtenir l'URL MCP">
    Copiez l'URL du serveur MCP : `docs.itellico.ai/mcp`
  </Step>

  <Step title="Configurer Votre Client IA">
    Ajoutez le serveur MCP à la configuration de votre application IA
  </Step>

  <Step title="Commencer à Utiliser">
    Votre IA peut maintenant rechercher et référencer directement la documentation itellicoAI
  </Step>
</Steps>

---

## Exemple d'Utilisation

Une fois connecté, votre assistant IA peut rechercher dans la documentation :

```
Utilisateur : "Comment créer un agent avec le SDK Python ?"
IA : [Utilise l'outil SearchItellicoAi]
IA : "Voici comment créer un agent avec le SDK Python..."
```

L'outil fournit un accès direct à :
- Références API
- Documentation SDK
- Guides de configuration
- Exemples de code
- Meilleures pratiques
