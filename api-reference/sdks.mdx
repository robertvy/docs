<AccordionGroup>
  <Accordion title="Type Safety" icon="shield-check">

## Python SDK

### Installation

<CodeGroup>
```bash pip
pip install itellicoai
```

```bash poetry
poetry add itellicoai
```
</CodeGroup>

### Requirements

- Python 3.8+
- Supports both sync and async operations

### Quick Start

```python
from itellicoai import ItellicoAI

# Initialize client
client = ItellicoAI(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    name="Customer Support Agent",
    model={
        "provider": "azure",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "21m00Tcm4TlvDq8ikWAM"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2"
    }
)

print(f"Created agent: {agent.id}")
```

### Async Support

```python
import asyncio
from itellicoai import AsyncItellicoAI

async def main():
    client = AsyncItellicoAI(api_key="your-api-key")

    # List agents asynchronously
    agents = await client.agents.list()

    for agent in agents:
        print(f"Agent: {agent.name}")

asyncio.run(main())
```

### Error Handling

```python
from itellicoai import ItellicoAI
from itellicoai.exceptions import (
    AuthenticationError,
    NotFoundError,
    RateLimitError,
    APIError
)

client = ItellicoAI(api_key="your-api-key")

try:
    agent = client.agents.retrieve("agent_123abc")
except AuthenticationError:
    print("Invalid API key")
except NotFoundError:
    print("Agent not found")
except RateLimitError:
    print("Rate limit exceeded")
except APIError as e:
    print(f"API error: {e.message}")
```

### Environment Variables

```python
import os
from itellicoai import ItellicoAI

# API key automatically loaded from ITELLICO_API_KEY env var
client = ItellicoAI()

# Or set explicitly
client = ItellicoAI(api_key=os.getenv("ITELLICO_API_KEY"))
```

### Error Handling

```typescript
import {
  ItellicoAI,
  AuthenticationError,
  NotFoundError,
  RateLimitError,
  APIError,
} from 'itellicoai';

const client = new ItellicoAI({ apiKey: 'your-api-key' });

try {
  const agent = await client.agents.retrieve('agent_123abc');
} catch (error) {
  if (error instanceof AuthenticationError) {
    console.error('Invalid API key');
  } else if (error instanceof NotFoundError) {
    console.error('Agent not found');
  } else if (error instanceof RateLimitError) {
    console.error('Rate limit exceeded');
  } else if (error instanceof APIError) {
    console.error(`API error: ${error.message}`);
  }
}
```

## SDK vs REST API Comparison

### Creating an Agent

<CodeGroup>
```python Python SDK
from itellicoai import ItellicoAI

client = ItellicoAI(api_key="your-api-key")

agent = client.agents.create(
    name="Support Agent",
    model={
        "provider": "azure",
        "model": "gpt-4o-mini"
    }
)
```

```typescript TypeScript SDK
import { ItellicoAI } from 'itellicoai';

const client = new ItellicoAI({ apiKey: 'your-api-key' });

const agent = await client.agents.create({
  name: 'Support Agent',
  model: {
    provider: 'azure',
    model: 'gpt-4o-mini',
  },
});
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {
      "provider": "azure",
      "model": "gpt-4o-mini"
    }
  }'
```
</CodeGroup>

### Benefits of SDK Approach

| Feature | SDK | REST API |
|---------|-----|----------|
| **Type Safety** | ✅ Full type hints/definitions | ❌ No built-in types |
| **Auto-completion** | ✅ IDE support | ❌ Manual reference needed |
| **Error Handling** | ✅ Structured exceptions | ⚠️ HTTP status codes only |
| **Authentication** | ✅ Automatic header management | ⚠️ Manual header setup |
| **Boilerplate** | ✅ Minimal | ❌ More verbose |
| **JSON Parsing** | ✅ Automatic | ⚠️ Manual |

## Best Practices

<AccordionGroup>
  <Accordion title="Use Environment Variables" icon="lock">
    Never hardcode API keys in your source code:

    ```bash
    # .env file
    ITELLICO_API_KEY=your-api-key
    ```

    Both SDKs automatically read from `ITELLICO_API_KEY` environment variable.
  </Accordion>

  <Accordion title="Handle Errors Gracefully" icon="shield">
    Always implement proper error handling for production applications:

    ```python
    try:
        agent = client.agents.create(...)
    except RateLimitError:
        # Implement exponential backoff
        time.sleep(60)
        agent = client.agents.create(...)
    except APIError as e:
        logger.error(f"API error: {e}")
        # Handle appropriately
    ```
  </Accordion>

  <Accordion title="Use Async for High Throughput" icon="bolt">
    For applications making many concurrent API calls, use async clients:

    ```python
    # Python
    from itellicoai import AsyncItellicoAI

    async def process_batch():
        client = AsyncItellicoAI()
        tasks = [client.agents.retrieve(id) for id in agent_ids]
        return await asyncio.gather(*tasks)
    ```
  </Accordion>

  <Accordion title="Leverage Type Safety" icon="check">
    Use TypeScript or Python type hints to catch errors at development time:

    ```typescript
    // TypeScript catches this error before runtime
    const agent = await client.agents.create({
      name: 'Agent',
      model: {
        provider: 'invalid',  // Error: Type '"invalid"' is not assignable
        model: 'gpt-4'
      }
    });
    ```
  </Accordion>
</AccordionGroup>
