## Getting Your API Key

Before using the SDKs, you'll need an API key from your itellicoAI dashboard.

<Steps>
  <Step title="Navigate to Developers">
    Go to your [itellicoAI dashboard](https://app.itellico.ai) and navigate to **Developers** from the sidebar.
  </Step>

  <Step title="Go to API Keys">
    Click on **API Keys** in the Developers section.
  </Step>

  <Step title="Create your key">
    Click the **Create API Key** button in the top-right corner.

    <Frame>
      <div className="block dark:hidden">
        <img src="/images/light.png" alt="API Keys dashboard showing how to create a new API key" />
      </div>
      <div className="hidden dark:block">
        <img src="/images/dark.png" alt="API Keys dashboard showing how to create a new API key" />
      </div>
    </Frame>
  </Step>

  <Step title="Copy and use with your SDK">
    Copy the generated API key and store it securely. Use this key when initializing the SDK.
  </Step>
</Steps>

<Warning>
Never share your API key or commit it to version control. Use environment variables to store your key securely.
</Warning>

## Quick Comparison

### Creating an Agent

<CodeGroup>
```python Python SDK
from itellicoai import Itellicoai

# Initialize client
client = Itellicoai(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    account_id="your-account-id",
    name="Customer Support Agent",
    model={
        "provider": "openai",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "EXAVITQu4vr4xnSDxMaL"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2:general",
        "language": "en-US"
    },
    initial_message={
        "mode": "fixed_message",
        "message": "Hello! How can I assist you today?",
        "delay_ms": 1000
    },
    max_duration_seconds=1800,
    tags=["support", "customer-service"]
)

print(f"Created agent: {agent.id}")
```

```typescript TypeScript SDK
import { Itellicoai } from 'itellicoai';

// Initialize client
const client = new Itellicoai({
  apiKey: 'your-api-key',
});

// Create an agent
const agent = await client.agents.create({
  accountId: 'your-account-id',
  name: 'Customer Support Agent',
  model: {
    provider: 'openai',
    model: 'gpt-4o-mini',
  },
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
  },
  transcriber: {
    provider: 'deepgram',
    model: 'nova-2:general',
    language: 'en-US',
  },
  initialMessage: {
    mode: 'fixed_message',
    message: 'Hello! How can I assist you today?',
    delayMs: 1000,
  },
  maxDurationSeconds: 1800,
  tags: ['support', 'customer-service'],
});

console.log(`Created agent: ${agent.id}`);
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {"provider": "azure", "model": "gpt-4o-mini"}
  }'
```
</CodeGroup>
