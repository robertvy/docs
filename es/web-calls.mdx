## Instalación

Agregar el widget a tu sitio es simple:

1. Crea y configura tu widget en el panel de itellicoAI
2. Copia el script de integración desde la pestaña Exportar
3. Pégalo antes de la etiqueta de cierre `</body>` en tu sitio web
4. El widget aparece automáticamente en tu sitio

```html
<!-- Pega esto antes de </body> -->
<script src="https://widget.itellico.ai/embed/YOUR-WIDGET-ID.js"></script>
```

<Note>
Funciona con cualquier plataforma de sitios web: WordPress, Shopify, Webflow, Wix, Squarespace, HTML personalizado y más.
</Note>
