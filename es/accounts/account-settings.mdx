## Información de la Cuenta

### Nombre de Cuenta

Cambia el nombre para mostrar de tu cuenta:

1. Ve a **Cuenta → Configuración**
2. Actualiza el campo **Nombre para Mostrar**
3. Haz clic en **Guardar**

<Tip>
Usa nombres descriptivos como "Acme Corp - Equipo de Ventas" en lugar de nombres genéricos como "Cuenta 1".
</Tip>

### Logo de Cuenta

Sube un logo para personalizar tu cuenta:

1. Ve a **Cuenta → Configuración**
2. Haz clic en **Subir Logo**
3. Selecciona un archivo de imagen
4. Haz clic en **Guardar**

**Requisitos:**
- Formato: PNG o JPEG
- Tamaño recomendado: 400×400px
- Tamaño máximo de archivo: 2MB

### ID de Cuenta

El identificador único (UUID) de tu cuenta se muestra en la página de configuración. Úsalo para solicitudes API y tickets de soporte.

**Para copiar:**
1. Ve a **Cuenta → Configuración**
2. Encuentra el campo **ID de Cuenta**
3. Haz clic en el botón copiar

<Note>
El ID de Cuenta no se puede cambiar. Se asigna automáticamente cuando se crea la cuenta.
</Note>
