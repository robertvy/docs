## Nombre del agente

El nombre del agente te ayuda a identificar este agente en tu panel de control, campañas e informes.

<Warning>
El nombre del agente es visible para los visitantes en los enlaces de demostración. Para nombres orientados al cliente, usa un nombre profesional como "Soporte Acme" o "Sarah de Ventas Acme" en lugar de etiquetas internas como "Calificador de leads v2".
</Warning>

### Mejores prácticas

**Para agentes orientados al cliente (visibles en demostraciones):**
- "Soporte Acme"
- "Alex de Ventas Acme"
- "Asistente de reservas"

**Para organización interna (múltiples agentes):**
- "Soporte al cliente - Preguntas sobre productos"
- "Ventas salientes - Calificación de leads"
- "Reserva de citas - Clínica dental"

**Evitar:**
- "Agente 1"
- "Prueba"
- "Nuevo agente"

**Usar nomenclatura consistente**
Desarrolla una convención de nomenclatura:
- `[Función] - [Especialidad]`
- `[Departamento] - [Tipo de agente]`
- `[Caso de uso] - [Idioma]`

<Note>
El avatar es visible cuando compartes enlaces de demostración, así que elige una imagen que represente tu marca profesionalmente.
</Note>

### Directrices de avatar

**Especificaciones de imagen**
- **Formato:** PNG, JPG, o SVG
- **Tamaño:** Mínimo 200x200px recomendado
- **Relación de aspecto:** Cuadrado (1:1)
- **Tamaño de archivo:** Menos de 2MB

### Estrategias de etiquetado

**Por función**
```text wrap
soporte, ventas, reservas, facturación, técnico
```

**Por idioma**
```text wrap
español, inglés, francés, alemán, multilingüe
```

**Por departamento**
```text wrap
servicio-cliente, equipo-ventas, operaciones, finanzas
```

**Por estado**
```text wrap
producción, prueba, archivado, beta
```

**Por región**
```text wrap
es, eu, us, apac, latam
```

### Usar etiquetas eficazmente

<Steps>
  <Step title="Crear un sistema de etiquetado">
    Definir categorías y convenciones antes de crear muchos agentes
  </Step>
  <Step title="Aplicar consistentemente">
    Usar las mismas etiquetas en agentes similares
  </Step>
  <Step title="Filtrar y buscar">
    Usar etiquetas para encontrar rápidamente grupos de agentes específicos en el panel de control
  </Step>
  <Step title="Revisar periódicamente">
    Limpiar etiquetas no utilizadas y actualizar según cambien las necesidades
  </Step>
</Steps>

## Mejores prácticas de organización

### Para equipos pequeños (1-5 agentes)

- Usar nombres simples y descriptivos
- Etiquetado mínimo necesario
- Notas básicas para cambios importantes

### Para equipos medianos (5-20 agentes)

- Desarrollar convenciones de nomenclatura
- Usar etiquetas para función y estado
- Mantener notas regulares sobre cambios

### Para equipos grandes (20+ agentes)

- Convenciones de nomenclatura estrictas obligatorias
- Etiquetado multinivel (función, región, idioma, estado)
- Registros de cambios detallados en notas
- Considerar crear documentación de agentes fuera de la plataforma
