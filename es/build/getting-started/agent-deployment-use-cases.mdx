## Cómo Funcionan los Agentes

Un agente de itellicoAI es una configuración IA única que puede implementarse en múltiples canales simultáneamente. El mismo agente puede:
- Manejar llamadas telefónicas entrantes
- Hacer llamadas salientes en campañas
- Integrarse en tu sitio web como widget
- Usar diferentes saludos para escenarios entrantes vs salientes

No necesariamente necesitas crear agentes "entrantes" o "salientes" separados - el mismo agente puede implementarse en múltiples canales.

## Implementación de Llamadas Telefónicas

Implementa tu agente para manejar conversaciones telefónicas.

### Llamadas Entrantes

<Steps>
  <Step title="Conecta un número telefónico">
    Conecta un número telefónico a tu agente vía troncal SIP
  </Step>
  <Step title="Configura saludo">
    Configura tu saludo entrante para llamadas entrantes
  </Step>
  <Step title="Activa">
    Tu agente está listo para responder cuando los clientes llamen
  </Step>
</Steps>

**Mejor para:**
- Líneas directas de soporte al cliente
- Consultas de estado de pedidos
- Líneas de soporte técnico
- Líneas de información general
- Programación de citas

**Ejemplo de saludo:**
```text wrap
Gracias por llamar a Soporte Acme. ¿Cómo puedo ayudarte hoy?
```

### Llamadas Salientes (Campañas)

<Steps>
  <Step title="Crea una campaña">
    Crea una nueva campaña y asigna tu agente
  </Step>
  <Step title="Agrega contactos">
    Sube tu lista de contactos y agrega contactos a la campaña
  </Step>
  <Step title="Configura saludo saliente">
    Configura la anulación de saludo saliente
  </Step>
  <Step title="Lanza campaña">
    Inicia tu campaña y monitorea resultados
  </Step>
</Steps>

**Mejor para:**
- Calificación de leads
- Recordatorios de citas
- Encuestas de retroalimentación de clientes
- Recordatorios de pago
- Llamadas de seguimiento

**Ejemplo de saludo:**
```text wrap
Hola, soy Alex de Acme Software. Recientemente solicitaste información sobre nuestro plan Enterprise. ¿Tienes un momento para conversar?
```

<Note>
El mismo agente puede manejar llamadas entrantes y salientes. Usa la característica de **anulación de saludo saliente** para personalizar cómo el agente se presenta en llamadas salientes.
</Note>

## Implementación de Widget Web

Integra tu agente en tu sitio web para interacciones basadas en navegador.

<Steps>
  <Step title="Crea un widget web">
    Crea un nuevo widget web en el panel
  </Step>
  <Step title="Asigna tu agente">
    Selecciona qué agente usar para este widget
  </Step>
  <Step title="Personaliza apariencia">
    Ajusta colores, posición y branding
  </Step>
  <Step title="Integra en tu sitio web">
    Copia el código de integración y agrégalo a tu sitio
  </Step>
  <Step title="Prueba y activa">
    Los visitantes ahora pueden usar voz o chat de texto
  </Step>
</Steps>

**Mejor para:**
- Soporte en vivo del sitio web
- Preguntas previas a la venta
- Captura de leads
- Demos de productos interactivos
- Asistencia web 24/7

**Características únicas:**
- Voz Y chat de texto en una interfaz
- No se requiere número telefónico
- Persiste en la navegación de páginas
- Puede cambiar entre voz y texto a mitad de conversación

## Configuración de Saludos

Dado que los agentes pueden usarse para llamadas entrantes y salientes, puedes configurar saludos por separado:

### Saludo Entrante
Usado cuando los clientes llaman a tu número telefónico o inician un chat web.

```text wrap
¡Hola! Gracias por llamar a Soporte Acme. ¿Cómo puedo ayudarte hoy?
```

### Saludo Saliente (Anulación)
Usado cuando el agente hace llamadas salientes en campañas.

```text wrap
Hola, soy Alex de Acme. Recientemente descargaste nuestra guía. ¿Tienes un momento rápido para discutir cómo podemos ayudarte?
```

<Tip>
Configura la anulación de saludo saliente en **Habilidades → Saludo → Anulación Saliente**. Si no está configurada, el agente usa el saludo entrante para todas las llamadas.
</Tip>
