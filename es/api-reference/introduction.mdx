## Autenticación

Todos los endpoints de la API requieren autenticación usando una clave API pasada en el encabezado `X-API-Key`:

```bash
curl -H "X-API-Key: your-api-key" https://api.itellico.ai/v1/accounts/current
```

<Info>
Aprende cómo crear y gestionar claves API en la [documentación de Claves API](/accounts/api-keys).
</Info>

## Recursos Principales

La API está organizada en torno a estos recursos principales:

- **Cuentas** - Gestiona tu cuenta y subcuentas
- **Agentes** - Crea y configura agentes de voz con IA
- **Proveedores** - Accede a modelos, transcriptores y voces disponibles
- **Números de Teléfono** - Gestiona números de teléfono para llamadas entrantes/salientes
- **Trunks SIP** - Configura trunks BYOC (Bring Your Own Carrier)
- **Conversaciones** - Accede al historial y detalles de conversaciones
- **Análisis** - Rastrea métricas de uso y datos de rendimiento
