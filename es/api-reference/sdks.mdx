## Obtener Tu Clave API

Antes de usar los SDKs, necesitarás una clave API desde tu panel de control de itellicoAI.

<Steps>
  <Step title="Navega a Desarrolladores">
    Ve a tu [panel de control de itellicoAI](https://app.itellico.ai) y navega a **Desarrolladores** desde la barra lateral.
  </Step>

  <Step title="Ve a Claves API">
    Haz clic en **Claves API** en la sección de Desarrolladores.
  </Step>

  <Step title="Crea tu clave">
    Haz clic en el botón **Crear Clave API** en la esquina superior derecha.

    <Frame>
      <div className="block dark:hidden">
        <img src="/images/light.png" alt="Panel de control de claves API mostrando cómo crear una nueva clave API" />
      </div>
      <div className="hidden dark:block">
        <img src="/images/dark.png" alt="Panel de control de claves API mostrando cómo crear una nueva clave API" />
      </div>
    </Frame>
  </Step>

  <Step title="Copia y usa con tu SDK">
    Copia la clave API generada y guárdala de forma segura. Usa esta clave al inicializar el SDK.
  </Step>
</Steps>

<Warning>
Nunca compartas tu clave API ni la subas al control de versiones. Usa variables de entorno para almacenar tu clave de forma segura.
</Warning>

## Comparación Rápida

### Crear un Agente

<CodeGroup>
```python Python SDK
from itellicoai import Itellicoai

# Initialize client
client = Itellicoai(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    account_id="your-account-id",
    name="Customer Support Agent",
    model={
        "provider": "openai",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "EXAVITQu4vr4xnSDxMaL"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2:general",
        "language": "en-US"
    },
    initial_message={
        "mode": "fixed_message",
        "message": "Hello! How can I assist you today?",
        "delay_ms": 1000
    },
    max_duration_seconds=1800,
    tags=["support", "customer-service"]
)

print(f"Created agent: {agent.id}")
```

```typescript TypeScript SDK
import { Itellicoai } from 'itellicoai';

// Initialize client
const client = new Itellicoai({
  apiKey: 'your-api-key',
});

// Create an agent
const agent = await client.agents.create({
  accountId: 'your-account-id',
  name: 'Customer Support Agent',
  model: {
    provider: 'openai',
    model: 'gpt-4o-mini',
  },
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
  },
  transcriber: {
    provider: 'deepgram',
    model: 'nova-2:general',
    language: 'en-US',
  },
  initialMessage: {
    mode: 'fixed_message',
    message: 'Hello! How can I assist you today?',
    delayMs: 1000,
  },
  maxDurationSeconds: 1800,
  tags: ['support', 'customer-service'],
});

console.log(`Created agent: ${agent.id}`);
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {"provider": "azure", "model": "gpt-4o-mini"}
  }'
```
</CodeGroup>
