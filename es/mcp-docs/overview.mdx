## Servidor MCP Alojado

Accede a tu servidor MCP y previsualiza las herramientas disponibles a través de nuestro servidor MCP de documentación alojado.

### URL del Servidor MCP

Utiliza la siguiente URL para conectar aplicaciones de IA a tu documentación:

```
docs.itellico.ai/mcp
```

---

## Herramientas Disponibles

### SearchItellicoAi

Busca en la base de conocimientos de itellicoAI para encontrar información relevante, ejemplos de código, referencias de API y guías.

**Usa esta herramienta cuando necesites:**
- Responder preguntas sobre itellicoAI
- Encontrar documentación específica
- Entender cómo funcionan las características
- Localizar detalles de implementación

**Devuelve:**
- Contenido contextual con títulos
- Enlaces directos a páginas de documentación

---

## Conectar Tu Aplicación de IA

<Steps>
  <Step title="Obtener la URL de MCP">
    Copia la URL del servidor MCP: `docs.itellico.ai/mcp`
  </Step>

  <Step title="Configurar Tu Cliente de IA">
    Añade el servidor MCP a la configuración de tu aplicación de IA
  </Step>

  <Step title="Comenzar a Usar">
    Tu IA ahora puede buscar y referenciar la documentación de itellicoAI directamente
  </Step>
</Steps>

---

## Ejemplo de Uso

Una vez conectado, tu asistente de IA puede buscar en la documentación:

```
Usuario: "¿Cómo creo un agente con el SDK de Python?"
IA: [Usa la herramienta SearchItellicoAi]
IA: "Así es cómo crear un agente con el SDK de Python..."
```

La herramienta proporciona acceso directo a:
- Referencias de API
- Documentación del SDK
- Guías de configuración
- Ejemplos de código
- Mejores prácticas
