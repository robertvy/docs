
## Authentifizierung

Alle API-Endpunkte erfordern eine Authentifizierung mittels eines API-Schlüssels, der im `X-API-Key`-Header übergeben wird:

```bash
curl -H "X-API-Key: your-api-key" https://api.itellico.ai/v1/accounts/current
```

## Wichtige Ressourcen

Die API ist um diese Hauptressourcen herum organisiert:

- **Accounts** - Verwalten Sie Ihr Konto und Unterkonten
- **Agents** - Erstellen und konfigurieren Sie KI-Voice-Agenten
- **Providers** - Zugriff auf verfügbare Modelle, Transcriber und Stimmen
- **Phone Numbers** - Verwalten Sie Telefonnummern für eingehende/ausgehende Anrufe
- **SIP Trunks** - Konfigurieren Sie BYOC (Bring Your Own Carrier)-Trunks
- **Conversations** - Zugriff auf Konversationsverlauf und Details
- **Analytics** - Verfolgen Sie Nutzungsmetriken und Leistungsdaten
