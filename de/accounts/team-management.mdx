## Team-Mitglieder einladen

Fügen Sie neue Benutzer zu Ihrem Konto hinzu, indem Sie E-Mail-Einladungen versenden.

<Steps>
  <Step title="Team-Einstellungen öffnen">
    Navigieren Sie zu **Einstellungen → Team** in Ihrem Konto
  </Step>

  <Step title="Auf Mitglieder einladen klicken">
    Klicken Sie auf die Schaltfläche **Mitglieder einladen**
  </Step>

  <Step title="E-Mail-Adressen eingeben">
    Fügen Sie eine oder mehrere E-Mail-Adressen hinzu (eine pro Zeile oder durch Komma getrennt)
  </Step>

  <Step title="Einladungen versenden">
    Klicken Sie auf **Einladungen versenden**

    Jede Person erhält eine E-Mail mit einem sicheren Link, um Ihrem Team beizutreten
  </Step>
</Steps>

### Einladungsdetails

**Ablauf:** Einladungen laufen nach **72 Stunden (3 Tage)** ab

**Die E-Mail enthält:**
- Wer sie eingeladen hat
- Kontoname
- Sicheren Akzeptanz-Link
- Ablaufdatum

## Einladungen annehmen

Wenn jemand eine Einladung erhält:

1. Die Person klickt auf den Link in der E-Mail
2. **Wenn sie kein Konto hat:** Sie meldet sich mit der eingeladenen E-Mail-Adresse an
3. **Wenn sie bereits ein Konto hat:** Sie meldet sich an und akzeptiert die Einladung
4. Sie wird Ihrem Team als Mitglied hinzugefügt
5. Das Konto erscheint in ihrem Konto-Umschalter

## Ausstehende Einladungen verwalten

Anzeigen und verwalten Sie von Ihnen gesendete Einladungen:

1. Gehen Sie zu **Einstellungen → Team**
2. Klicken Sie auf die Registerkarte **Ausstehende Einladungen**

**Sie können:**
- **Erneut senden** - Eine neue Einladungs-E-Mail mit einer neuen 72-Stunden-Gültigkeit senden
- **Abbrechen** - Die Einladung löschen (der Link wird ungültig)

<Tip>
Wenn jemand die Einladung nicht erhält, überprüfen Sie dessen Spam-Ordner oder senden Sie sie erneut.
</Tip>

## Team-Mitglieder anzeigen

Die Team-Seite zeigt alle Mitglieder mit:
- Name und E-Mail
- Status (Aktiv oder Ausstehende Einladung)
- Datum der letzten Anmeldung
- Beitrittsdatum

**Suchen und filtern:**
- Suche nach Name oder E-Mail
- Filter nach Status (Aktiv vs. Ausstehend)

## Team-Mitglieder entfernen

<Warning>
Entfernte Mitglieder verlieren sofort den Zugriff.
</Warning>

<Steps>
  <Step title="Zu den Team-Einstellungen gehen">
    Navigieren Sie zu **Einstellungen → Team**
  </Step>

  <Step title="Mitglied finden">
    Suchen Sie das Team-Mitglied, das Sie entfernen möchten
  </Step>

  <Step title="Entfernen">
    Klicken Sie auf das Menü (⋯) neben dem Namen → **Aus Team entfernen**
  </Step>

  <Step title="Bestätigen">
    Bestätigen Sie die Entfernung

    Die Person kann nicht mehr auf dieses Konto zugreifen
  </Step>
</Steps>

**Was passiert:**
- ❌ Mitglied verliert sofort den Zugriff
- ❌ Konto verschwindet aus dem Konto-Umschalter
- ✅ Ihre erstellten Inhalte bleiben erhalten (Agenten, Konfigurationen)
- ✅ Sie können bei Bedarf erneut eingeladen werden

## Best Practices

<AccordionGroup>
  <Accordion title="Onboarding neuer Mitglieder" icon="user-plus">
    **Wenn jemand beitritt:**
    1. Geben Sie ihnen eine Tour durch Ihre Agenten
    2. Teilen Sie relevante Dokumentation
    3. Setzen Sie klare Erwartungen für ihre Rolle
    4. Melden Sie sich nach der ersten Woche
  </Accordion>

  <Accordion title="Einladungen verwalten" icon="envelope">
    **Tipps:**
    - Überprüfen Sie E-Mail-Adressen vor dem Versenden doppelt
    - Folgen Sie nach, wenn nach 2 Tagen keine Antwort erfolgt
    - Senden Sie erneut, wenn die Einladung abläuft
    - Brechen Sie Einladungen ab, die an falsche Adressen gesendet wurden
    - Räumen Sie monatlich abgelaufene Einladungen auf
  </Accordion>

  <Accordion title="Offboarding" icon="user-minus">
    **Wenn jemand geht:**
    1. Entfernen Sie sofort den Zugriff
    2. Widerrufen Sie alle von ihnen erstellten API-Schlüssel
    3. Aktualisieren Sie gemeinsam genutzte Zugangsdaten, auf die sie Zugriff hatten
    4. Dokumentieren Sie ihre Arbeit für die Übergabe
  </Accordion>

  <Accordion title="Externe Mitarbeiter" icon="user-tie">
    **Für Auftragnehmer oder Partner:**
    - Erwägen Sie ein separates Unterkonto für Kunden
    - Setzen Sie Kalendererinnerungen zur Überprüfung des Zugriffs
    - Entfernen Sie den Zugriff, wenn das Engagement endet
  </Accordion>
</AccordionGroup>

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Einladung nicht erhalten" icon="envelope">
    **Häufige Ursachen:**
    - E-Mail im Spam-Ordner
    - Tippfehler in der E-Mail-Adresse
    - E-Mail vom Server des Empfängers blockiert

    **Lösungen:**
    1. Bitten Sie sie, den Spam-Ordner zu überprüfen
    2. Überprüfen Sie, ob die E-Mail-Adresse korrekt ist
    3. Senden Sie die Einladung erneut
  </Accordion>

  <Accordion title="Einladung abgelaufen" icon="clock">
    **Lösung:**
    1. Gehen Sie zu **Einstellungen → Team → Ausstehende Einladungen**
    2. Finden Sie die abgelaufene Einladung
    3. Klicken Sie auf **Erneut senden**
    4. Eine neue 72-Stunden-Einladung wird gesendet
  </Accordion>

  <Accordion title="Kann ein Mitglied nicht entfernen" icon="ban">
    **Mögliche Ursachen:**
    - Sie haben keine Berechtigung
    - Das Mitglied kann nicht entfernt werden

    **Lösung:**
    Kontaktieren Sie Ihren Kontoadministrator, wenn Sie Hilfe benötigen.
  </Accordion>

  <Accordion title="Mitglied kann das Konto nach dem Annehmen nicht sehen" icon="eye-slash">
    **Häufige Ursachen:**
    - Benutzer hat sich mit einer anderen E-Mail angemeldet
    - Konto nach der Einladung deaktiviert

    **Lösungen:**
    - Überprüfen Sie, ob sie die eingeladene E-Mail-Adresse verwendet haben
    - Lassen Sie sie alle Konten im Konto-Umschalter überprüfen
    - Überprüfen Sie, ob das Konto noch aktiv ist
  </Accordion>
</AccordionGroup>

## Sicherheitstipps

<AccordionGroup>
  <Accordion title="E-Mail-Adressen überprüfen" icon="shield-check">
    Vor dem Einladen:
    - Überprüfen Sie die Rechtschreibung doppelt
    - Überprüfen Sie, ob die Domain korrekt ist (achten Sie auf .co vs. .com)
    - Verwenden Sie geschäftliche E-Mails für Unternehmenskonten
  </Accordion>

  <Accordion title="Zugriffsprüfungen" icon="calendar-check">
    Überprüfen Sie Team-Mitglieder vierteljährlich:
    - Überprüfen Sie, ob jede Person noch Zugriff benötigt
    - Entfernen Sie inaktive Benutzer
    - Dokumentieren Sie die Überprüfung
  </Accordion>

  <Accordion title="Austritts-Checkliste" icon="list-check">
    Wenn jemand geht:
    - [ ] Sofort aus allen Konten entfernen
    - [ ] Alle von ihnen erstellten API-Schlüssel widerrufen
    - [ ] Gemeinsame Passwörter ändern, die sie kannten
    - [ ] Ihre jüngsten Aktivitäten überprüfen
  </Accordion>
</AccordionGroup>

## Häufig gestellte Fragen

<AccordionGroup>

  <Accordion title="Wie lange sind Einladungen gültig?">
    Einladungen laufen nach **72 Stunden (3 Tage)** ab. Sie können abgelaufene Einladungen erneut senden, um einen neuen Link zu generieren.
  </Accordion>

  <Accordion title="Kann ich eine persönliche Nachricht mit der Einladung senden?">
    Derzeit nicht. Einladungen werden mit einer Standard-E-Mail-Vorlage gesendet.
  </Accordion>

  <Accordion title="Wie viele Personen kann ich einladen?">
    Die Teamgröße hängt von Ihrem Abonnementplan ab. Überprüfen Sie Ihre aktuellen Limits unter **Einstellungen → Konto**.
  </Accordion>

  <Accordion title="Kann jemand eine Einladung annehmen, nachdem sie abgelaufen ist?">
    Nein. Abgelaufene Einladungslinks funktionieren nicht. Senden Sie die Einladung erneut, um einen neuen gültigen Link zu generieren.
  </Accordion>
</AccordionGroup>
