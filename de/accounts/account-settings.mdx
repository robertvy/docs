## Kontoinformationen

### Kontoname

Ändere den Anzeigenamen deines Kontos:

1. Gehe zu **Konto → Einstellungen**
2. Aktualisiere das Feld **Anzeigename**
3. Klicke auf **Speichern**

<Tip>
Verwende beschreibende Namen wie "Acme Corp - Vertriebsteam" anstelle von generischen Namen wie "Konto 1".
</Tip>

### Konto-Logo

Lade ein Logo hoch, um dein Konto zu personalisieren:

1. Gehe zu **Konto → Einstellungen**
2. Klicke auf **Logo hochladen**
3. Wähle eine Bilddatei aus
4. Klicke auf **Speichern**

**Anforderungen:**
- Format: PNG oder JPEG
- Empfohlene Größe: 400×400px
- Maximale Dateigröße: 2MB

### Konto-ID

Die eindeutige Kennung (UUID) deines Kontos wird auf der Einstellungsseite angezeigt. Verwende sie für API-Anfragen und Support-Tickets.

**Zum Kopieren:**
1. Gehe zu **Konto → Einstellungen**
2. Finde das Feld **Konto-ID**
3. Klicke auf die Schaltfläche zum Kopieren

<Note>
Die Konto-ID kann nicht geändert werden. Sie wird automatisch zugewiesen, wenn das Konto erstellt wird.
</Note>
