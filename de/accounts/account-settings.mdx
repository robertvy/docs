## Kontoinformationen

### Kontoname

Ändern Sie den Anzeigenamen Ihres Kontos:

1. Gehen Sie zu **Konto → Einstellungen**
2. Aktualisieren Sie das Feld **Anzeigename**
3. Klicken Sie auf **Speichern**

<Tip>
Verwenden Sie beschreibende Namen wie "Acme Corp - Vertriebsteam" anstelle von generischen Namen wie "Konto 1".
</Tip>

### Konto-Logo

Laden Sie ein Logo hoch, um Ihr Konto zu personalisieren:

1. Gehen Sie zu **Konto → Einstellungen**
2. Klicken Sie auf **Logo hochladen**
3. Wählen Sie eine Bilddatei aus
4. Klicken Sie auf **Speichern**

**Anforderungen:**
- Format: PNG oder JPEG
- Empfohlene Größe: 400×400px
- Maximale Dateigröße: 2MB

### Konto-ID

Die eindeutige Kennung (UUID) Ihres Kontos wird auf der Einstellungsseite angezeigt. Verwenden Sie sie für API-Anfragen und Support-Tickets.

**Zum Kopieren:**
1. Gehen Sie zu **Konto → Einstellungen**
2. Finden Sie das Feld **Konto-ID**
3. Klicken Sie auf die Schaltfläche zum Kopieren

<Note>
Die Konto-ID kann nicht geändert werden. Sie wird automatisch zugewiesen, wenn das Konto erstellt wird.
</Note>
