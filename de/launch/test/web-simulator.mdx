## Web-Simulator-Tests

Der Web-Simulator bietet sofortige, browserbasierte Tests Ihres KI-Agenten mit der gleichen Sprachtechnologie wie bei Produktionsanrufen.

**Hauptfunktionen**:
- Testen Sie Gespräche ohne Telefonanrufe
- Echtzeit-Transkript- und Aktionsüberwachung
- Vorab ausgefüllte Testvariablen für dynamischen Kontext
- Vollständige Protokollierung in Conversations

## Schnellstart

1. Öffnen Sie Ihren Agenten im Dashboard
2. Der Simulator erscheint auf der rechten Seite
3. Klicken Sie auf **Test starten**, um zu beginnen
4. Erlauben Sie den Mikrofonzugriff, wenn Sie dazu aufgefordert werden
5. Sprechen Sie natürlich mit Ihrem Agenten

## Was zu testen ist

- **Begrüßung**: Überprüfen Sie Aussprache und Klarheit
- **Anweisungen**: Bestätigen Sie, dass der Agent Ihre Regeln befolgt
- **Aktionen**: Lösen Sie benutzerdefinierte APIs, Weiterleitungen, Buchungen aus
- **Wissen**: Stellen Sie Fragen, die in Ihrer Wissensdatenbank beantwortet werden
- **Unterbrechungen**: Testen Sie das Barge-in-Verhalten
- **Grenzfälle**: Stille, undeutliche Sprache, unerwartete Anfragen
