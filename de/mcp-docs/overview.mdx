**Beta-Feature** - Unser MCP-Server befindet sich derzeit in der Beta-Phase. Verbinde KI-Anwendungen direkt mit der itellicoAI-Dokumentation.
</Note>

## Gehosteter MCP-Server

Greife auf deinen MCP-Server zu und zeige verfügbare Tools über unseren gehosteten Dokumentations-MCP-Server in der Vorschau an.

### MCP-Server-URL

Verwende die folgende URL, um KI-Anwendungen mit deiner Dokumentation zu verbinden:

```
docs.itellico.ai/mcp
```

---

## Verfügbare Tools

### SearchItellicoAi

Durchsuche die itellicoAI-Wissensdatenbank, um relevante Informationen, Codebeispiele, API-Referenzen und Anleitungen zu finden.

**Verwende dieses Tool, wenn du:**
- Fragen zu itellicoAI beantworten musst
- Spezifische Dokumentation finden möchtest
- Verstehen möchtest, wie Features funktionieren
- Implementierungsdetails suchst

**Gibt zurück:**
- Kontextbezogene Inhalte mit Titeln
- Direkte Links zu Dokumentationsseiten

---

## Verbinde deine KI-Anwendung

<Steps>
  <Step title="Hole dir die MCP-URL">
    Kopiere die MCP-Server-URL: `docs.itellico.ai/mcp`
  </Step>

  <Step title="Konfiguriere deinen KI-Client">
    Füge den MCP-Server zur Konfiguration deiner KI-Anwendung hinzu
  </Step>

  <Step title="Beginne mit der Nutzung">
    Deine KI kann jetzt direkt die itellicoAI-Dokumentation durchsuchen und referenzieren
  </Step>
</Steps>

---

## Beispielverwendung

Nach der Verbindung kann dein KI-Assistent die Dokumentation durchsuchen:

```
Benutzer: "Wie erstelle ich einen Agenten mit dem Python SDK?"
KI: [Verwendet SearchItellicoAi-Tool]
KI: "So erstellen Sie einen Agenten mit dem Python SDK..."
```

Das Tool bietet direkten Zugriff auf:
- API-Referenzen
- SDK-Dokumentation
- Setup-Anleitungen
- Codebeispiele
- Best Practices
