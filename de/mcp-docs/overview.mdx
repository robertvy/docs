## Gehosteter MCP-Server

Greifen Sie auf Ihren MCP-Server zu und zeigen Sie verfügbare Tools über unseren gehosteten Dokumentations-MCP-Server in der Vorschau an.

### MCP-Server-URL

Verwenden Sie die folgende URL, um KI-Anwendungen mit Ihrer Dokumentation zu verbinden:

```
docs.itellico.ai/mcp
```

---

## Verfügbare Tools

### SearchItellicoAi

Durchsuchen Sie die itellicoAI-Wissensdatenbank, um relevante Informationen, Codebeispiele, API-Referenzen und Anleitungen zu finden.

**Verwenden Sie dieses Tool, wenn Sie:**
- Fragen zu itellicoAI beantworten müssen
- Spezifische Dokumentation finden möchten
- Verstehen möchten, wie Features funktionieren
- Implementierungsdetails suchen

**Gibt zurück:**
- Kontextbezogene Inhalte mit Titeln
- Direkte Links zu Dokumentationsseiten

---

## Verbinden Sie Ihre KI-Anwendung

<Steps>
  <Step title="Holen Sie sich die MCP-URL">
    Kopieren Sie die MCP-Server-URL: `docs.itellico.ai/mcp`
  </Step>

  <Step title="Konfigurieren Sie Ihren KI-Client">
    Fügen Sie den MCP-Server zur Konfiguration Ihrer KI-Anwendung hinzu
  </Step>

  <Step title="Beginnen Sie mit der Nutzung">
    Ihre KI kann jetzt direkt die itellicoAI-Dokumentation durchsuchen und referenzieren
  </Step>
</Steps>

---

## Beispielverwendung

Nach der Verbindung kann Ihr KI-Assistent die Dokumentation durchsuchen:

```
Benutzer: "Wie erstelle ich einen Agenten mit dem Python SDK?"
KI: [Verwendet SearchItellicoAi-Tool]
KI: "So erstellen Sie einen Agenten mit dem Python SDK..."
```

Das Tool bietet direkten Zugriff auf:
- API-Referenzen
- SDK-Dokumentation
- Setup-Anleitungen
- Codebeispiele
- Best Practices
