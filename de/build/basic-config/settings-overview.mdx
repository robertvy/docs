## Models-Tab

Wählen Sie die AI-Technologien aus und konfigurieren Sie diese, die Ihren Agent antreiben.

### Model (LLM)
Wählen Sie das Sprachmodell für das Reasoning.

**Anbieter:**
- Azure OpenAI
- OpenAI
- Anthropic
- Groq

**Funktionen:**
- Modellauswahl mit EU-Hosting-Badges
- Temperaturkontrolle

[Mehr über AI-Modelle erfahren →](/build/voice-speech/choose-ai-model)

### Voice
Wählen Sie aus, wie Ihr Agent klingen soll.

**Anbieter:**
- ElevenLabs
- Azure
- Cartesia

**Funktionen:**
- Stimmbibliothek mit Vorschauen
- Stimmkonfiguration (variiert je nach Anbieter)

[Mehr über Stimmauswahl erfahren →](/build/voice-speech/select-voice)

### Transcriber
Konfigurieren Sie Speech-to-Text.

**Anbieter:**
- Deepgram
- Azure Speech

**Funktionen:**
- Sprachauswahl und mehrsprachige Unterstützung
- Keyword-Boosting

[Mehr über Transkription erfahren →](/build/voice-speech/transcriber)


### Sound Effects
Fügen Sie Umgebungsaudio zu Gesprächen hinzu.
- Hintergrund-Ambiente (Büro, Café, Straße)

[Mehr über Ambient Sound erfahren →](/build/voice-speech/ambient-sound)

- Variableneinfügung

## Navigationstipps

### Verwendung des Konfigurationspanels

<Steps>
  <Step title="Models konfigurieren">
    Wählen Sie AI-Modell, Stimme und Transcriber
  </Step>
  <Step title="Abilities einrichten">
    Konfigurieren Sie Anweisungen, Begrüßungen und Wissen
  </Step>
  <Step title="Operations feinabstimmen">
    Passen Sie VAD- und Timeout-Einstellungen nach dem Testen an
  </Step>
  <Step title="Analytics einrichten">
    Definieren Sie Ziele zur Messung
  </Step>
</Steps>

### Speicherverhalten

Einige Abschnitte (wie Modellauswahl) speichern automatisch. Andere erfordern das Klicken auf die Schaltfläche Speichern. Änderungen gelten nur für zukünftige Anrufe, nicht für laufende Anrufe.
