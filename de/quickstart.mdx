Erstellen Sie einen KI-Sprachagenten, der Kundengespräche über Telefon und Web abwickelt. Keine Programmierung erforderlich - folgen Sie einfach diesen Schritten.

  </Card>
  <Card title="Gespräche testen" icon="vial">

  </Card>

## Bevor Sie beginnen

<Accordion title="Was Sie benötigen" icon="list-check">
- itellicoAI-Konto ([hier registrieren](https://app.itellico.ai/register))
- Klarer Anwendungsfall (Support, Buchung, Vertrieb usw.)
- 5 Minuten Zeit
</Accordion>

## Schritt 1: Erstellen Sie Ihren Agenten

<Steps>
  <Step title="Navigieren Sie zu KI-Agenten">
    Öffnen Sie das itellicoAI-Dashboard und klicken Sie in der Seitenleiste auf **KI-Agenten**
  </Step>
  <Step title="Neuen Agenten erstellen">
    Klicken Sie auf **Agent erstellen** und geben Sie einen beschreibenden Namen ein wie "Kundensupport-Assistent"
  </Step>
  <Step title="Agenten-Builder öffnen">
    Klicken Sie auf **Erstellen**, um die Agentenkonfigurationsoberfläche zu öffnen
  </Step>
</Steps>

<Tip>
Wählen Sie Namen, welche die Funktion Ihres Agenten widerspiegeln, für eine einfachere Verwaltung beim Skalieren.
</Tip>

## Schritt 2: Konfigurieren Sie Ihren Agenten

### Anweisungen festlegen

Definieren Sie, wie sich Ihr Agent verhalten soll:

1. Erweitern Sie **Fähigkeiten** und klicken Sie auf **Anweisungen**
2. Verwenden Sie diese Vorlage als Ausgangspunkt:

```
Sie sind die professionelle Stimme von [Ihr Firmenname].

Ihre Ziele:
- Hilfreiche, genaue Informationen bereitstellen
- Einfache Anfragen eigenständig lösen
- Komplexe Probleme an menschliche Agenten eskalieren

Kommunikationsstil:
- Professionell, aber freundlich
- Antworten unter 30 Sekunden halten
- Klare, umsetzbare Anleitung geben
```

### Begrüßung konfigurieren

Richten Sie den ersten Eindruck ein:

1. Wählen Sie unter **Fähigkeiten** die Option **Begrüßung**
2. Wählen Sie **KI-Agent beginnt mit fester Nachricht**
3. Geben Sie Ihre Begrüßung ein:

```
Hallo! Vielen Dank für Ihre Kontaktaufnahme mit [Firmenname]. Wie kann ich Ihnen heute helfen?
```

4. Setzen Sie die Verzögerung auf **0,5 Sekunden** für natürliches Timing

### KI-Modell & Stimme auswählen

Wählen Sie die Technologie, die Ihren Agenten antreibt:

1. Erweitern Sie **Modelle** und klicken Sie auf **Modell**
2. Wählen Sie **Azure OpenAI** mit **GPT-4.1 Mini** (empfohlen)
3. Klicken Sie auf **Stimme** und wählen Sie **ElevenLabs**
4. Hören Sie sich Stimmen an und wählen Sie eine, die zu Ihrer Marke passt
5. Wählen Sie unter **Transkription** **Deepgram Nova-3 General** mit Ihrer bevorzugten Sprache oder mehrsprachig

## Schritt 3: Testen Sie Ihren Agenten

Bevor Sie live gehen, testen Sie Ihren Agenten:

1. Klicken Sie auf **Test Call** (obere rechte Ecke)
2. Klicken Sie auf **Start Web Call**
3. Erteilen Sie Mikrofon-Berechtigungen, wenn Sie dazu aufgefordert werden
4. Probieren Sie verschiedene Szenarien aus:
   - "Hallo, ich brauche Hilfe mit meinem Konto"
   - "Können Sie mir etwas über Ihre Dienstleistungen erzählen?"
   - "Ich möchte mit jemandem sprechen"
5. Bewerten Sie Antworten auf Klarheit und Ton

<Note>
Wenn die Antworten nicht passen, passen Sie Ihre Anweisungen an und testen Sie erneut. Iteration ist der Schlüssel zu einem großartigen Agenten.
</Note>

## Schritt 4: Stellen Sie Ihren Agenten bereit

Wählen Sie, wie Sie Ihren Agenten bereitstellen möchten:

<Tabs>
  <Tab title="Telefonanrufe">
    **Machen Sie Ihren Agenten per Telefon verfügbar**

    1. Navigieren Sie zu **Telefonie → SIP-Trunks** im Dashboard
    2. Klicken Sie auf **SIP-Trunk erstellen** und konfigurieren Sie Ihre Telefonieanbieter-Einstellungen
    3. Gehen Sie zu **Telefonie → Telefonnummern**
    4. Klicken Sie auf **Telefonnummer erstellen** und verknüpfen Sie sie mit Ihrem SIP-Trunk
    5. Weisen Sie Ihren Agenten der Telefonnummer zu

    Ihr Agent ist jetzt live und kann eingehende Anrufe bearbeiten!

    <Note>
    Sie haben keinen SIP-Trunk? Senden Sie eine E-Mail an [support@itellico.ai](mailto:support@itellico.ai), um die Bereitstellung von Telefonnummern anzufordern.
    </Note>

    <Card title="Telefon-Integrationsleitfaden" icon="phone" href="/de/phone-calls">
      Erfahren Sie mehr über SIP-Trunk-Einrichtung und Telefonbereitstellung
    </Card>
  </Tab>

  <Tab title="Web-Widget">
    **Betten Sie Ihren Agenten auf Ihrer Website ein**

    1. Navigieren Sie zu **Web → Widgets**
    2. Klicken Sie auf **Widget erstellen** und wählen Sie Ihren Agenten aus
    3. Passen Sie Erscheinungsbild und Verhalten an
    4. Kopieren Sie das Einbettungsskript aus dem **Export**-Tab
    5. Fügen Sie es vor `</body>` auf Ihrer Website ein

    ```html
    <script src="https://widget.itellico.ai/embed/IHRE-WIDGET-ID.js"></script>
    ```

    Ihr Agent ist jetzt live auf Ihrer Website mit Sprache und Chat!

    <Card title="Widget-Einrichtungsleitfaden" icon="globe" href="/de/web-calls">
      Erfahren Sie mehr über Widget-Anpassung
    </Card>
  </Tab>
</Tabs>
