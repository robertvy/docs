## Überblick

Fügen Sie KI-gestützte Sprach- und Chatgespräche zu Ihrer Website mit dem itellicoAI-Web-Widget hinzu. Besucher können mit Ihrem KI-Agenten über ihre Stimme sprechen oder Nachrichten in einer Chat-Oberfläche eingeben - beides in einem einzigen, anpassbaren Widget, das auf jeder Website eingebettet werden kann.

## So funktioniert es

<Steps>
  <Step title="Widget im Dashboard erstellen">
    Wählen Sie Ihren KI-Agenten und konfigurieren Sie das Erscheinungsbild und Verhalten des Widgets
  </Step>
  <Step title="Auf Ihrer Website einbetten">
    Kopieren Sie das Einbettungsskript und fügen Sie es in Ihre Website ein
  </Step>
  <Step title="Besucher interagieren">
    Website-Besucher können Sprachanrufe oder Textgespräche mit Ihrer KI starten
  </Step>
  <Step title="Leistung überwachen">
    Verfolgen Sie Gespräche und optimieren Sie Ihren Agenten basierend auf echten Interaktionen
  </Step>
</Steps>

## Widget-Konfiguration

Passen Sie Ihr Widget über das itellicoAI-Dashboard an:

- **Erscheinungsbild** - Wählen Sie Farben, Layout, Avatar und Animationen
- **Inhalte** - Passen Sie alle Nachrichten und Beschriftungen in mehreren Sprachen an
- **Funktionen** - Schalten Sie Sprach-/Chat-Modi, Transkripte und Feedback-Sammlung um
- **Aktionen** - Fügen Sie anklickbare Schaltflächen für Links oder Telefonnummern hinzu
- **Datenschutz** - Konfigurieren Sie Einwilligungsbanner und rechtliche Texte
- **Domains** - Beschränken Sie, wo das Widget eingebettet werden kann

## Installation

Das Hinzufügen des Widgets zu Ihrer Website ist einfach:

1. Erstellen und konfigurieren Sie Ihr Widget im itellicoAI-Dashboard
2. Kopieren Sie das Einbettungsskript aus dem Export-Tab
3. Fügen Sie es vor dem schließenden `</body>`-Tag auf Ihrer Website ein
4. Das Widget erscheint automatisch auf Ihrer Website

```html
<!-- Fügen Sie dies vor </body> ein -->
<script src="https://widget.itellico.ai/embed/IHRE-WIDGET-ID.js"></script>
```

<Note>
Funktioniert mit jeder Website-Plattform: WordPress, Shopify, Webflow, Wix, Squarespace, benutzerdefiniertes HTML und mehr.
</Note>
