## Überblick

Füge KI-gestützte Sprach- und Chatgespräche zu deiner Website mit dem itellicoAI-Web-Widget hinzu. - **Erscheinungsbild** - Wähle Farben, Layout, Avatar und Animationen
- **Inhalte** - Passe alle Nachrichten und Beschriftungen in mehreren Sprachen an
- **Funktionen** - Schalte Sprach-/Chat-Modi, Transkripte und Feedback-Sammlung um
- **Aktionen** - Füge anklickbare Schaltflächen für Links oder Telefonnummern hinzu
- **Datenschutz** - Konfiguriere Einwilligungsbanner und rechtliche Texte
- **Domains** - Beschränken du, wo das Widget eingebettet werden kann

## Installation

Das Hinzufügen des Widgets zu deiner Website ist einfach:

1. Erstelle und konfiguriere dein Widget im itellicoAI-Dashboard
2. Kopiere das Einbettungsskript aus dem Export-Tab
3. Füge es vor dem schließenden `</body>`-Tag auf deiner Website ein
4. Das Widget erscheint automatisch auf deiner Website

```html
<!-- Füge dies vor </body> ein -->
<script src="https://widget.itellico.ai/embed/DEINE-WIDGET-ID.js"></script>
```

<Note>
Funktioniert mit jeder Website-Plattform: WordPress, Shopify, Webflow, Wix, Squarespace, benutzerdefiniertes HTML und mehr.
</Note>
