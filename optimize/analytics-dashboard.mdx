## Overview

The itellicoAI analytics dashboard provides real-time insights into agent performance, conversation outcomes, and operational metrics. Use these tools to measure success, identify improvement opportunities, and optimize your voice AI operations.

**Access:** Navigate to **Analytics → Conversations** in the dashboard

## Key Metrics Overview

### High-Level Performance

The dashboard displays key performance indicators at a glance:

**Conversation Volume:**
- Total conversations (inbound + outbound)
- Conversations per agent
- Daily/weekly/monthly trends
- Peak call times

**Conversation Outcomes:**
- Goal achievement rate
- Transfer rate
- Average duration
- Completion rate

**Action Metrics:**
- Total actions triggered
- Action success rate
- Most used actions
- Failed action count

**Cost Tracking:**
- Total cost per period
- Cost per conversation
- Cost by agent
- Cost trends

## Filtering and Analysis

### Date Range Filtering

Select custom date ranges to analyze specific periods:

<Steps>
  <Step title="Click date selector">
    At the top of the analytics dashboard
  </Step>

  <Step title="Choose range">
    Options include:
    - Today
    - Yesterday
    - Last 7 days
    - Last 30 days
    - This month
    - Last month
    - Custom range
  </Step>

  <Step title="Apply filter">
    All metrics update automatically
  </Step>
</Steps>

**Use cases for date filtering:**
- Compare before/after changes
- Analyze launch day performance
- Monthly reporting
- Identify seasonal patterns

---

### Agent Filtering

Filter metrics by specific agent(s):

**Single agent view:**
- Isolate performance of one agent
- Useful for testing new configurations
- Compare against historical performance

**Multi-agent comparison:**
- Side-by-side agent performance
- Identify top performers
- Spot outliers
- Benchmark similar agents

**All agents view:**
- Organization-wide metrics
- Total operational performance
- Aggregate trends

---

### Phone Number Filtering

Filter conversations by phone number:

**By caller number:**
- Track specific customer interactions
- Analyze VIP customer experience
- Identify problematic callers

**By agent phone number:**
- Compare performance across different lines
- Analyze geographic patterns
- Test number routing

## Conversation Goal Analytics

Track and analyze conversation goal completion across your agents.

### Goal Dashboard

**Access:** **Analytics → Goals** or filter in Conversations

**Metrics displayed:**
- Total goals evaluated
- Goals achieved count
- Goals not achieved count
- Achievement rate (%)
- Trends over time

### Goal Performance by Agent

Compare goal achievement across agents:

| Agent | Total Calls | Goals Achieved | Achievement Rate |
|-------|-------------|----------------|------------------|
| Support Agent | 1,247 | 1,089 | 87.3% |
| Sales Agent | 892 | 756 | 84.8% |
| Booking Agent | 2,103 | 1,998 | 95.0% |

### Goal Reasoning Analysis

For each conversation, view:
- Goal name and criteria
- Achievement status (✅ Achieved / ❌ Not achieved)
- AI-generated reasoning explaining the outcome
- Relevant conversation excerpts

**Example goal reasoning:**
```
Goal: "Resolve customer issue"
Status: ✅ Achieved
Reasoning: "Customer stated their password reset problem
was successfully resolved. Agent sent reset email and
customer confirmed receipt. Call ended with customer
thanking the agent."
```

**Use goal reasoning to:**
- Understand why goals failed
- Identify edge cases
- Improve goal definitions
- Train team on successful patterns

## Exporting Data

### Conversation Exports

Export detailed conversation data for external analysis.

<Steps>
  <Step title="Navigate to Conversations">
    Go to **Conversations → Logs**
  </Step>

  <Step title="Apply filters">
    Filter by agent, date range, status, etc.

    Only export the conversations you need
  </Step>

  <Step title="Click Export">
    Look for the Export button at the top of the conversation list
  </Step>

  <Step title="Select format">
    Choose export format:
    - **CSV** - For spreadsheets and BI tools
    - **JSON** - For programmatic analysis
  </Step>

  <Step title="Select data fields">
    Choose what to include:
    - Basic metadata (date, duration, agent)
    - Full transcripts
    - Action payloads and responses
    - Goal results and reasoning
    - Post-call analysis responses
    - Custom data fields
    - Cost data
  </Step>

  <Step title="Download">
    File downloads immediately

    Large exports may take a few moments
  </Step>
</Steps>

**Exported data includes:**
- Conversation ID
- Date and time
- Agent name and ID
- Phone numbers (caller and agent)
- Duration
- Status (completed, failed, etc.)
- Full transcript with timestamps
- Actions triggered with full payloads
- Goal evaluation results
- Post-call analysis answers
- Cost breakdown
- Custom metadata

### CSV Export Structure

```csv
conversation_id,date,agent_name,duration_seconds,status,goal_achieved,goal_reasoning,primary_issue,customer_satisfied,cost_total
conv_abc123,2025-01-15 10:30:00,Support Agent,142,completed,true,"Customer issue resolved...",password_reset,yes,0.23
conv_def456,2025-01-15 10:35:00,Support Agent,89,completed,false,"Customer hung up...",billing,no,0.15
```

### JSON Export Structure

```json
[
  {
    "conversation_id": "conv_abc123",
    "date": "2025-01-15T10:30:00Z",
    "agent": {
      "id": "agent_123",
      "name": "Support Agent"
    },
    "duration_seconds": 142,
    "status": "completed",
    "transcript": [
      {"speaker": "agent", "text": "Hello, how can I help?", "timestamp": 0},
      {"speaker": "user", "text": "I need to reset my password", "timestamp": 3}
    ],
    "actions": [
      {
        "name": "send_password_reset",
        "status": "success",
        "parameters": {"email": "user@example.com"},
        "response": {"success": true}
      }
    ],
    "goals": [
      {
        "name": "Resolve customer issue",
        "achieved": true,
        "reasoning": "Customer confirmed issue resolved..."
      }
    ],
    "post_call_analysis": {
      "customer_satisfied": "yes",
      "followed_procedures": "yes",
      "primary_issue": "password_reset"
    },
    "cost": {
      "total": 0.23,
      "llm": 0.15,
      "tts": 0.05,
      "stt": 0.03
    }
  }
]
```

## Campaign Reporting Exports

For outbound campaigns, export comprehensive campaign performance data.

### Campaign Analytics

**Access:** **Campaigns → [Your Campaign] → Analytics**

**Metrics available:**
- Total calls attempted
- Calls completed
- Calls failed (with failure reasons)
- Average call duration
- Goal achievement rate
- Action success rate
- Cost per call
- ROI metrics (if configured)

### Campaign Export

<Steps>
  <Step title="Open campaign">
    Navigate to the specific campaign
  </Step>

  <Step title="Go to Analytics tab">
    Click the Analytics or Reporting tab
  </Step>

  <Step title="Click Export Report">
    Export button typically at top-right
  </Step>

  <Step title="Choose level of detail">
    Options:
    - **Summary** - High-level campaign metrics
    - **Detailed** - Per-call breakdown
    - **Full** - Includes transcripts and analysis
  </Step>

  <Step title="Download">
    Campaign report downloads as CSV or Excel
  </Step>
</Steps>

**Campaign export includes:**
- Campaign name and dates
- Contact list details
- Call outcomes (completed, failed, no answer, voicemail, etc.)
- Conversation duration per call
- Goals achieved per call
- Actions executed per call
- Post-call analysis per call
- Cost per call and total campaign cost
- Timestamps for each call attempt

### Campaign Performance Report Example

```
Campaign: January Sales Outreach
Period: 2025-01-01 to 2025-01-31
Agent: Sales Agent

Summary:
- Total contacts: 1,000
- Calls completed: 847 (84.7%)

- Goals achieved: 623 (73.6% of completed)

- Interested (meeting booked): 312 (36.8%)
```

## Using Exports for Analysis

### Import into BI Tools

Export data integrates with:
- **Excel/Google Sheets** - Pivot tables, charts
- **Tableau/Power BI** - Advanced visualizations
- **SQL Databases** - Long-term storage and querying
- **Python/R** - Statistical analysis
- **Custom dashboards** - Build your own reporting

### Common Analysis Workflows

<AccordionGroup>
  <Accordion title="Monthly reporting" icon="calendar">
    1. Export all conversations for the month
    2. Import into Excel/Sheets
    3. Create pivot tables for key metrics
    4. Generate charts for stakeholder presentations
    5. Compare to previous months
  </Accordion>

  <Accordion title="Agent performance comparison" icon="users">
    1. Export conversations filtered by date range
    2. Group by agent in analysis tool
    3. Compare metrics:
       - Goal achievement rates
       - Average duration
       - Transfer rates
       - Customer satisfaction
    4. Identify top performers and training opportunities
  </Accordion>

  <Accordion title="Cost analysis" icon="dollar-sign">
    1. Export with cost data included
    2. Analyze cost drivers:
       - Which agents are most expensive?
       - Which conversations cost most?
       - Where can we optimize?
    3. Calculate ROI for different use cases
    4. Project future costs based on trends
  </Accordion>

  <Accordion title="Quality assurance" icon="shield-check">
    1. Export random sample of conversations
    2. Review post-call analysis responses
    3. Listen to audio for sample calls
    4. Score quality on custom criteria
    5. Identify training and improvement needs
  </Accordion>

  <Accordion title="Trend analysis" icon="chart-line">

    2. Plot metrics over time

## Best Practices

<AccordionGroup>
  <Accordion title="Export regularly" icon="calendar-check">
    Schedule regular exports for archival:
    - Weekly for active monitoring
    - Monthly for reporting
    - Quarterly for trend analysis

    Don't rely solely on dashboard—export for long-term records
  </Accordion>

  <Accordion title="Use consistent date ranges" icon="ruler">
    Compare apples to apples:
    - Same day of week (avoid Mon vs Fri comparisons)
    - Same time ranges
    - Account for holidays/seasonality

    Makes trend analysis more accurate
  </Accordion>

  <Accordion title="Combine with qualitative review" icon="headphones">
    Numbers don't tell the full story:
    - Listen to sample calls
    - Read transcripts
    - Understand context behind metrics

    Quantitative + qualitative = complete picture
  </Accordion>

  <Accordion title="Share insights widely" icon="share-nodes">
    Don't silo analytics:
    - Share reports with stakeholders
    - Present trends to team
    - Use data for training
    - Celebrate wins publicly

    Data-driven culture improves outcomes
  </Accordion>

  <Accordion title="Act on insights" icon="rocket">
    Analytics without action is wasted:
    - Set improvement targets based on data
    - Test changes and measure impact
    - Iterate based on results
    - Close the feedback loop

    Measurement → Insight → Action → Improvement
  </Accordion>
</AccordionGroup>
