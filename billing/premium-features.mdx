## Overview

Some premium providers and features incur additional per-minute or per-use charges on top of your base usage rate. These surcharges reflect the higher costs of premium AI capabilities.

---

## Premium Language Models

Using advanced language models adds a per-minute surcharge:

| Model | Surcharge | Notes |
|-------|-----------|-------|
| GPT-4.1 Mini | €0.00 | **Included in base** |
| GPT-4.1 Nano | €0.00 | **Included in base** |
| Claude Haiku 4.5 | €0.00 | **Included in base** |
| GPT-4.1 | +€0.02/min | More capable |
| GPT-4o | +€0.02/min | Multimodal |
| GPT-5 Mini | +€0.02/min | Next-gen efficient |
| GPT-5 | +€0.04/min | Flagship |
| GPT-5.1 | +€0.05/min | Latest flagship |
| Claude Sonnet 4.5 | +€0.04/min | Advanced reasoning |
| Claude Opus 4 | +€0.07/min | Most capable |

<Note>
Surcharges are calculated per minute of call duration, not per API call.
</Note>

## Per-Call Charges

Some features are charged per call rather than per minute:

### Outbound Calling

| Fee | Rate | Notes |
|-----|------|-------|
| Outbound call attempt | €0.015/attempt | Charged per dial, regardless of outcome |
| Batch/Campaign dial | €0.01/dial | Discounted for campaign calls |
| Branded Caller ID | €0.08/call | Display custom caller name |

### Post-Call Processing

| Feature | Rate | Notes |
|---------|------|-------|
| Post-call analysis | €0.05/call | AI summary, sentiment, action items |
| Premium analysis | €0.12/call | Deep analysis with Claude Opus |
| Transcript export | €0.00 | Included |

## Knowledge Base Charges

Large knowledge bases incur monthly charges:

| Size | Monthly Fee | Notes |
|------|-------------|-------|
| Up to 10MB | €0.00 | Included in plan limits |
| 10-50MB | €5.00/KB | Per knowledge base |
| 50-200MB | €15.00/KB | Per knowledge base |
| 200MB+ | €35.00/KB | Per knowledge base |

Additionally, a **€0.005/min** surcharge applies when a knowledge base is actively queried during a call.

## Example: Premium Stack Pricing

**Scenario:** Using ElevenLabs + Claude Opus for a high-quality sales agent

| Component | Rate |
|-----------|------|
| Base rate (Pro tier) | €0.13/min |
| ElevenLabs surcharge | +€0.02/min |
| Claude Opus surcharge | +€0.07/min |
| **Total** | **€0.22/min** |

**For a 5-minute call:** 5 × €0.22 = €1.10
