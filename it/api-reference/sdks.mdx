## Ottenere la Tua Chiave API

Prima di utilizzare gli SDK, avrai bisogno di una chiave API dal tuo pannello di controllo itellicoAI.

<Steps>
  <Step title="Vai a Sviluppatori">
    Vai al tuo [pannello di controllo itellicoAI](https://app.itellico.ai) e naviga su **Sviluppatori** dalla barra laterale.
  </Step>

  <Step title="Vai a Chiavi API">
    Clicca su **Chiavi API** nella sezione Sviluppatori.
  </Step>

  <Step title="Crea la tua chiave">
    Clicca sul pulsante **Crea Chiave API** nell'angolo in alto a destra.

    <Frame>
      <div className="block dark:hidden">
        <img src="/images/light.png" alt="Pannello delle chiavi API che mostra come creare una nuova chiave API" />
      </div>
      <div className="hidden dark:block">
        <img src="/images/dark.png" alt="Pannello delle chiavi API che mostra come creare una nuova chiave API" />
      </div>
    </Frame>
  </Step>

  <Step title="Copia e usa con il tuo SDK">
    Copia la chiave API generata e conservala in modo sicuro. Usa questa chiave durante l'inizializzazione dell'SDK.
  </Step>
</Steps>

<Warning>
Non condividere mai la tua chiave API e non caricarla nel controllo di versione. Usa variabili d'ambiente per memorizzare la tua chiave in modo sicuro.
</Warning>

## Confronto Rapido

### Creare un Agente

<CodeGroup>
```python Python SDK
from itellicoai import Itellicoai

# Initialize client
client = Itellicoai(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    account_id="your-account-id",
    name="Customer Support Agent",
    model={
        "provider": "openai",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "EXAVITQu4vr4xnSDxMaL"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2:general",
        "language": "en-US"
    },
    initial_message={
        "mode": "fixed_message",
        "message": "Hello! How can I assist you today?",
        "delay_ms": 1000
    },
    max_duration_seconds=1800,
    tags=["support", "customer-service"]
)

print(f"Created agent: {agent.id}")
```

```typescript TypeScript SDK
import { Itellicoai } from 'itellicoai';

// Initialize client
const client = new Itellicoai({
  apiKey: 'your-api-key',
});

// Create an agent
const agent = await client.agents.create({
  accountId: 'your-account-id',
  name: 'Customer Support Agent',
  model: {
    provider: 'openai',
    model: 'gpt-4o-mini',
  },
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
  },
  transcriber: {
    provider: 'deepgram',
    model: 'nova-2:general',
    language: 'en-US',
  },
  initialMessage: {
    mode: 'fixed_message',
    message: 'Hello! How can I assist you today?',
    delayMs: 1000,
  },
  maxDurationSeconds: 1800,
  tags: ['support', 'customer-service'],
});

console.log(`Created agent: ${agent.id}`);
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {"provider": "azure", "model": "gpt-4o-mini"}
  }'
```
</CodeGroup>
