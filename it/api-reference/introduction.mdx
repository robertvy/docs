## Autenticazione

Tutti gli endpoint API richiedono autenticazione utilizzando una chiave API passata nell'header `X-API-Key`:

```bash
curl -H "X-API-Key: your-api-key" https://api.itellico.ai/v1/accounts/current
```

<Info>
Scopri come creare e gestire le chiavi API nella [documentazione delle Chiavi API](/accounts/api-keys).
</Info>

## Risorse Principali

L'API √® organizzata attorno a queste risorse principali:

- **Account** - Gestisci il tuo account e sottoconti
- **Agenti** - Crea e configura agenti vocali IA
- **Provider** - Accedi a modelli, trascrittori e voci disponibili
- **Numeri di Telefono** - Gestisci numeri di telefono per chiamate in entrata/uscita
- **Trunk SIP** - Configura trunk BYOC (Bring Your Own Carrier)
- **Conversazioni** - Accedi alla cronologia e ai dettagli delle conversazioni
- **Analisi** - Monitora metriche di utilizzo e dati sulle prestazioni
