## Installazione

Aggiungere il widget al tuo sito è semplice:

1. Crea e configura il tuo widget nella dashboard itellicoAI
2. Copia lo script di incorporamento dalla scheda Esporta
3. Incollalo prima del tag di chiusura `</body>` sul tuo sito web
4. Il widget appare automaticamente sul tuo sito

```html
<!-- Incolla questo prima di </body> -->
<script src="https://widget.itellico.ai/embed/YOUR-WIDGET-ID.js"></script>
```

<Note>
Funziona con qualsiasi piattaforma di sito web: WordPress, Shopify, Webflow, Wix, Squarespace, HTML personalizzato e altro.
</Note>
