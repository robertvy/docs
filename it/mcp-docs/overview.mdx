## Server MCP Ospitato

Accedi al tuo server MCP e visualizza in anteprima gli strumenti disponibili tramite il nostro server MCP di documentazione ospitato.

### URL del Server MCP

Utilizza il seguente URL per connettere applicazioni IA alla tua documentazione:

```
docs.itellico.ai/mcp
```

---

## Strumenti Disponibili

### SearchItellicoAi

Cerca nella base di conoscenze itellicoAI per trovare informazioni rilevanti, esempi di codice, riferimenti API e guide.

**Usa questo strumento quando hai bisogno di:**
- Rispondere a domande su itellicoAI
- Trovare documentazione specifica
- Capire come funzionano le funzionalità
- Localizzare i dettagli di implementazione

**Restituisce:**
- Contenuto contestuale con titoli
- Link diretti alle pagine di documentazione

---

## Collegare la Tua Applicazione IA

<Steps>
  <Step title="Ottenere l'URL MCP">
    Copia l'URL del server MCP: `docs.itellico.ai/mcp`
  </Step>

  <Step title="Configurare il Tuo Client IA">
    Aggiungi il server MCP alla configurazione della tua applicazione IA
  </Step>

  <Step title="Iniziare a Usare">
    La tua IA può ora cercare e fare riferimento direttamente alla documentazione itellicoAI
  </Step>
</Steps>

---

## Esempio di Utilizzo

Una volta connesso, il tuo assistente IA può cercare nella documentazione:

```
Utente: "Come creo un agente con l'SDK Python?"
IA: [Usa lo strumento SearchItellicoAi]
IA: "Ecco come creare un agente con l'SDK Python..."
```

Lo strumento fornisce accesso diretto a:
- Riferimenti API
- Documentazione SDK
- Guide di configurazione
- Esempi di codice
- Migliori pratiche
