## Test Simulatore Web

Il simulatore web fornisce test istantaneo basato su browser del tuo agente AI utilizzando la stessa tecnologia vocale delle chiamate di produzione.

**Funzionalità Chiave**:
- Testa conversazioni senza chiamate telefoniche
- Monitoraggio trascrizione e azioni in tempo reale
- Pre-popola variabili test per contesto dinamico
- Logging completo in Conversations

## Avvio Rapido

1. Apri il tuo agente nella dashboard
2. Il simulatore appare sul lato destro
3. Clicca **Start Test** per iniziare
4. Permetti accesso microfono quando richiesto
5. Parla naturalmente con il tuo agente

## Cosa Testare

- **Saluto**: Verifica pronuncia e chiarezza
- **Istruzioni**: Conferma che l'agente segua le tue regole
- **Azioni**: Attiva API personalizzate, trasferimenti, prenotazioni
- **Knowledge**: Fai domande a cui si risponde nella tua KB
- **Interruzioni**: Testa comportamento barge-in
- **Casi limite**: Silenzio, parlato poco chiaro, richieste inaspettate
