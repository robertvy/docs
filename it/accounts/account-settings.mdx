## Informazioni Account

### Nome Account

Modifica il nome visualizzato del tuo account:

1. Vai in **Account → Impostazioni**
2. Aggiorna il campo **Nome Visualizzato**
3. Clicca **Salva**

<Tip>
Usa nomi descrittivi come "Acme Corp - Team Vendite" invece di nomi generici come "Account 1".
</Tip>

### Logo Account

Carica un logo per personalizzare il tuo account:

1. Vai in **Account → Impostazioni**
2. Clicca **Carica Logo**
3. Seleziona un file immagine
4. Clicca **Salva**

**Requisiti:**
- Formato: PNG o JPEG
- Dimensione consigliata: 400×400px
- Dimensione massima file: 2MB

### ID Account

L'identificatore univoco (UUID) del tuo account viene visualizzato nella pagina delle impostazioni. Usalo per richieste API e ticket di supporto.

**Per copiare:**
1. Vai in **Account → Impostazioni**
2. Trova il campo **ID Account**
3. Clicca il pulsante di copia

<Note>
L'ID Account non può essere modificato. Viene assegnato automaticamente alla creazione dell'account.
</Note>
