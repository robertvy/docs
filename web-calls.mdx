## Getting Started

<Steps>
  <Step title="Install SDK">
    Add the itellicoAI web SDK to your application
    ```bash
    npm install @itellico/web-sdk
    ```
  </Step>
  <Step title="Initialize client">
    Set up the web call client with your API credentials
    ```javascript
    import { WebCallClient } from '@itellico/web-sdk';

    const client = new WebCallClient({
      apiKey: 'your-api-key'
    });
    ```
  </Step>
  <Step title="Start a call">
    Initiate a voice conversation from your web app
    ```javascript
    await client.startCall({
      agentId: 'your-agent-id'
    });
    ```
  </Step>
  <Step title="Customize experience">
    Configure UI elements and conversation flows
  </Step>
</Steps>

## Integration Examples

<CodeGroup>
```javascript React
import { useWebCall } from '@itellico/web-sdk/react';

function VoiceAssistant() {
  const { startCall, endCall, isActive } = useWebCall({
    apiKey: process.env.ITELLICO_API_KEY,
    agentId: 'assistant-1'
  });

  return (
    <button onClick={isActive ? endCall : startCall}>
      {isActive ? 'End Call' : 'Start Voice Chat'}
    </button>
  );
}
```

```html JavaScript
<script src="https://cdn.itellico.ai/web-sdk.js"></script>
<script>
  const client = new ItellicaAI.WebCallClient({
    apiKey: 'your-api-key'
  });

  document.getElementById('call-button').onclick = () => {
    client.startCall({ agentId: 'assistant-1' });
  };
</script>
```
</CodeGroup>
