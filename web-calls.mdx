## Overview

Add AI-powered voice and chat conversations to your website with the itellicoAI web widget. Visitors can talk to your AI agent using their voice or type messages in a chat interface - both in a single, customizable widget that embeds on any site.

## Installation

Adding the widget to your site is simple:

1. Create and configure your widget in the itellicoAI dashboard
2. Copy the embed script from the Export tab
3. Paste it before the closing `</body>` tag on your website
4. The widget appears automatically on your site

```html
<!-- Paste this before </body> -->
<script src="https://widget.itellico.ai/embed/YOUR-WIDGET-ID.js"></script>
```

<Note>
Works with any website platform: WordPress, Shopify, Webflow, Wix, Squarespace, custom HTML, and more.
</Note>
