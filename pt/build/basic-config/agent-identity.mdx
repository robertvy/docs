## Nome do Agente

O nome do agente ajuda você a identificar este agente no seu painel, campanhas e relatórios.

<Warning>
O nome do agente é visível aos visitantes em links de demonstração. Para nomes voltados ao cliente, use um nome profissional como "Suporte Acme" ou "Sarah da Vendas Acme" em vez de rótulos internos como "Qualificador de Leads v2".
</Warning>

### Melhores Práticas

**Para agentes voltados ao cliente (visíveis em demonstrações):**
- "Suporte Acme"
- "Alex da Vendas Acme"
- "Assistente de Agendamento"

**Para organização interna (múltiplos agentes):**
- "Suporte ao Cliente - Perguntas sobre Produtos"
- "Vendas de Saída - Qualificação de Leads"
- "Agendamento de Consultas - Clínica Odontológica"

**Evite:**
- "Agente 1"
- "Teste"
- "Novo Agente"

**Use Nomenclatura Consistente**
Desenvolva uma convenção de nomenclatura:
- `[Função] - [Especialidade]`
- `[Departamento] - [Tipo de Agente]`
- `[Caso de Uso] - [Idioma]`

<Note>
O avatar é visível quando você compartilha links de demonstração, então escolha uma imagem que represente sua marca profissionalmente.
</Note>

### Diretrizes para Avatar

**Especificações da Imagem**
- **Formato:** PNG, JPG ou SVG
- **Tamanho:** Recomendado 200x200px mínimo
- **Proporção:** Quadrada (1:1)
- **Tamanho do Arquivo:** Menos de 2MB

### Estratégias de Marcação

**Por Função**
```text wrap
suporte, vendas, agendamento, cobrança, técnico
```

**Por Idioma**
```text wrap
português, espanhol, francês, alemão, multilíngue
```

**Por Departamento**
```text wrap
atendimento-cliente, equipe-vendas, operações, financeiro
```

**Por Status**
```text wrap
produção, teste, arquivado, beta
```

**Por Região**
```text wrap
br, eua, ue, apac, latam
```

### Usando Tags Efetivamente

<Steps>
  <Step title="Criar um sistema de marcação">
    Defina categorias e convenções antes de criar muitos agentes
  </Step>
  <Step title="Aplicar consistentemente">
    Use as mesmas tags em agentes semelhantes
  </Step>
  <Step title="Filtrar e pesquisar">
    Use tags para encontrar rapidamente grupos específicos de agentes no painel
  </Step>
  <Step title="Revisar periodicamente">
    Limpe tags não utilizadas e atualize conforme as necessidades mudam
  </Step>
</Steps>

## Notas Internas

Use a seção de notas para documentar detalhes de configuração, histórico de versões e informações da equipe sobre este agente.

### Tipos Úteis de Notas

**Histórico de Configuração**
```text wrap
2025-01-15: Mudou para GPT-4.1 Mini para respostas mais rápidas
2025-01-10: Adicionada integração Cal.com para agendamento
2025-01-05: Atualizada base de conhecimento com novas informações de produto
```

**Problemas Conhecidos**
```text wrap
- Ocasionalmente tem dificuldade com sotaques fortes (ajustando configurações VAD)
- Transferência para vendas às vezes cai às sextas-feiras (investigando)
```

**Responsabilidades da Equipe**
```text wrap
Proprietário: Sarah (Equipe de Produto)
Backup: Mike (Líder de Suporte)
Revisões: Mensalmente na primeira segunda-feira
```

**Notas de Desempenho**
```text wrap
Duração média de chamada: 3m 45s
Taxa de resolução: 87%
Satisfação do cliente: 4.6/5
Última otimização: 2025-01-10
```

## Melhores Práticas de Organização

### Para Equipes Pequenas (1-5 agentes)

- Use nomes simples e descritivos
- Marcação mínima necessária
- Notas básicas para mudanças importantes

### Para Equipes Médias (5-20 agentes)

- Desenvolva convenções de nomenclatura
- Use tags para função e status
- Mantenha notas regulares sobre mudanças

### Para Equipes Grandes (20+ agentes)

- Convenções de nomenclatura estritas obrigatórias
- Marcação multi-nível (função, região, idioma, status)
- Registros detalhados de mudanças nas notas
- Considere criar documentação de agentes fora da plataforma
