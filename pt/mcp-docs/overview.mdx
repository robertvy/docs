## Servidor MCP Hospedado

Acesse seu servidor MCP e visualize as ferramentas disponíveis através do nosso servidor MCP de documentação hospedado.

### URL do Servidor MCP

Use a seguinte URL para conectar aplicações de IA à sua documentação:

```
docs.itellico.ai/mcp
```

---

## Ferramentas Disponíveis

### SearchItellicoAi

Pesquise na base de conhecimento itellicoAI para encontrar informações relevantes, exemplos de código, referências de API e guias.

**Use esta ferramenta quando precisar:**
- Responder perguntas sobre itellicoAI
- Encontrar documentação específica
- Entender como os recursos funcionam
- Localizar detalhes de implementação

**Retorna:**
- Conteúdo contextual com títulos
- Links diretos para páginas de documentação

---

## Conectando Sua Aplicação de IA

<Steps>
  <Step title="Obter a URL do MCP">
    Copie a URL do servidor MCP: `docs.itellico.ai/mcp`
  </Step>

  <Step title="Configurar Seu Cliente de IA">
    Adicione o servidor MCP à configuração da sua aplicação de IA
  </Step>

  <Step title="Começar a Usar">
    Sua IA agora pode pesquisar e referenciar a documentação itellicoAI diretamente
  </Step>
</Steps>

---

## Exemplo de Uso

Uma vez conectado, seu assistente de IA pode pesquisar a documentação:

```
Usuário: "Como criar um agente com o SDK Python?"
IA: [Usa a ferramenta SearchItellicoAi]
IA: "Aqui está como criar um agente com o SDK Python..."
```

A ferramenta fornece acesso direto a:
- Referências de API
- Documentação do SDK
- Guias de configuração
- Exemplos de código
- Melhores práticas
