## Obter Sua Chave API

Antes de usar os SDKs, você precisará de uma chave API do seu painel de controle itellicoAI.

<Steps>
  <Step title="Navegue até Desenvolvedores">
    Vá para seu [painel de controle itellicoAI](https://app.itellico.ai) e navegue até **Desenvolvedores** na barra lateral.
  </Step>

  <Step title="Vá para Chaves API">
    Clique em **Chaves API** na seção Desenvolvedores.
  </Step>

  <Step title="Crie sua chave">
    Clique no botão **Criar Chave API** no canto superior direito.

    <Frame>
      <div className="block dark:hidden">
        <img src="/images/light.png" alt="Painel de chaves API mostrando como criar uma nova chave API" />
      </div>
      <div className="hidden dark:block">
        <img src="/images/dark.png" alt="Painel de chaves API mostrando como criar uma nova chave API" />
      </div>
    </Frame>
  </Step>

  <Step title="Copie e use com seu SDK">
    Copie a chave API gerada e armazene-a com segurança. Use esta chave ao inicializar o SDK.
  </Step>
</Steps>

<Warning>
Nunca compartilhe sua chave API ou a envie para o controle de versão. Use variáveis de ambiente para armazenar sua chave com segurança.
</Warning>

## Comparação Rápida

### Criar um Agente

<CodeGroup>
```python Python SDK
from itellicoai import Itellicoai

# Initialize client
client = Itellicoai(
    api_key="your-api-key"
)

# Create an agent
agent = client.agents.create(
    account_id="your-account-id",
    name="Customer Support Agent",
    model={
        "provider": "openai",
        "model": "gpt-4o-mini"
    },
    voice={
        "provider": "elevenlabs",
        "voice_id": "EXAVITQu4vr4xnSDxMaL"
    },
    transcriber={
        "provider": "deepgram",
        "model": "nova-2:general",
        "language": "en-US"
    },
    initial_message={
        "mode": "fixed_message",
        "message": "Hello! How can I assist you today?",
        "delay_ms": 1000
    },
    max_duration_seconds=1800,
    tags=["support", "customer-service"]
)

print(f"Created agent: {agent.id}")
```

```typescript TypeScript SDK
import { Itellicoai } from 'itellicoai';

// Initialize client
const client = new Itellicoai({
  apiKey: 'your-api-key',
});

// Create an agent
const agent = await client.agents.create({
  accountId: 'your-account-id',
  name: 'Customer Support Agent',
  model: {
    provider: 'openai',
    model: 'gpt-4o-mini',
  },
  voice: {
    provider: 'elevenlabs',
    voiceId: 'EXAVITQu4vr4xnSDxMaL',
  },
  transcriber: {
    provider: 'deepgram',
    model: 'nova-2:general',
    language: 'en-US',
  },
  initialMessage: {
    mode: 'fixed_message',
    message: 'Hello! How can I assist you today?',
    delayMs: 1000,
  },
  maxDurationSeconds: 1800,
  tags: ['support', 'customer-service'],
});

console.log(`Created agent: ${agent.id}`);
```

```bash REST API
curl -X POST https://api.itellico.ai/v1/agents \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "model": {"provider": "azure", "model": "gpt-4o-mini"}
  }'
```
