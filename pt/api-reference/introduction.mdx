## Autenticação

Todos os endpoints da API requerem autenticação usando uma chave API passada no cabeçalho `X-API-Key`:

```bash
curl -H "X-API-Key: your-api-key" https://api.itellico.ai/v1/accounts/current
```

<Info>
Aprenda como criar e gerenciar chaves API na [documentação de Chaves API](/accounts/api-keys).
</Info>

## Recursos Principais

A API está organizada em torno destes recursos principais:

- **Contas** - Gerencie sua conta e subcontas
- **Agentes** - Crie e configure agentes de voz com IA
- **Provedores** - Acesse modelos, transcritores e vozes disponíveis
- **Números de Telefone** - Gerencie números de telefone para chamadas de entrada/saída
- **Trunks SIP** - Configure trunks BYOC (Bring Your Own Carrier)
- **Conversas** - Acesse histórico e detalhes de conversas
- **Análises** - Rastreie métricas de uso e dados de desempenho
