## Web Simulator Testing

The web simulator provides instant, browser-based testing of your AI agent using the same voice technology as production calls.

**Key Features**:
- Test conversations without phone calls
- Real-time transcript and action monitoring
- Pre-populate test variables for dynamic context
- Full logging in Conversations

## Quick Start

1. Open your agent in the dashboard
2. The simulator appears on the right side
3. Click **Start Test** to begin
4. Allow microphone access when prompted
5. Speak naturally with your agent

## What to Test

- **Greeting**: Verify pronunciation and clarity
- **Instructions**: Confirm agent follows your rules
- **Actions**: Trigger custom APIs, transfers, bookings
- **Knowledge**: Ask questions answered in your KB
- **Interruptions**: Test barge-in behavior
- **Edge cases**: Silence, unclear speech, unexpected requests
