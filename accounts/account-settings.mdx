## Account Information

### Account Name

Change your account's display name:

1. Go to **Account → Settings**
2. Update the **Display Name** field
3. Click **Save**

<Tip>
Use descriptive names like "Acme Corp - Sales Team" instead of generic names like "Account 1".
</Tip>

### Account Logo

Upload a logo to personalize your account:

1. Go to **Account → Settings**
2. Click **Upload Logo**
3. Select an image file
4. Click **Save**

**Requirements:**
- Format: PNG or JPEG
- Recommended size: 400×400px
- Max file size: 2MB

### Account ID

Your account's unique identifier (UUID) is displayed on the settings page. Use it for API requests and support tickets.

**To copy:**
1. Go to **Account → Settings**
2. Find the **Account ID** field
3. Click the copy button

<Note>
The Account ID cannot be changed. It's automatically assigned when the account is created.
</Note>
